**Type:** GET

**Description:** `/future/copytrade/user/v1/copy-trade/follower-leaders-page`

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| type | number | No | - | Type (1: current follow, 2: history) | 1, 2 |
| direction | string | No | - | Direction | NEXT, PREV |
| limit | number | No | - | Page size | 10 |
| id | number | No | - | ID | - |

---

### Limit Flow Rules

2/s/apikey

---
