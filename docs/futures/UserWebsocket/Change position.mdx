### Subscribe

```json title="Subscribe"
{
  "method": "SUBSCRIBE",
  "params": ["position@{listenKey}"],
  "id": "{id}"
}
```

### Unsubscribe

```json title="Unsubscribe"
{
  "method": "UNSUBSCRIBE",
  "params": ["position@{listenKey}"],
  "id": "{id}"
}
```

---
