### Message Format

```json title="Format"
{
  "topic": "trade",
  "event": "trade@btc_usdt", // title
  "data": {}
}
```

---

### Example: Transaction Record (Real-time Push Message)

```json title="Example"
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "i": 6316559590087222000,
    "t": 1655992403617,
    "p": "43000",
    "q": "0.21",
    "b": true
  }
}
```
