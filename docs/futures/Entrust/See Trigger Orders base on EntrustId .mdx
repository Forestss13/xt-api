**Type:** GET

**Description:** `/future/trade/v1/entrust/plan-detail`

### Limit Flow Rules

200/s/apikey

### Parameters

| Name      | Type    | Mandatory | Default | Description | Ranges |
| --------- | ------- | --------- | ------- | ----------- | ------ |
| entrustId | integer | true      | N/A     | Order ID    |        |

### Response Example

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "closePosition": false, // Whether triggered to close all
    "createdTime": 0, // Create time
    "entrustId": 0, // Order ID
    "entrustType": "", // Order type
    "marketOrderLevel": 0, // Best market price
    "orderSide": "", // Order side
    "ordinary": true,
    "origQty": 0, // Quantity (Cont)
    "positionSide": "", // Position side
    "price": 0, // Order price
    "state": "", // Order state: NOT_TRIGGERED; TRIGGERING; TRIGGERED; USER_REVOCATION; PLATFORM_REVOCATION; EXPIRED; UNFINISHED; HISTORY
    "stopPrice": 0, // Trigger price
    "symbol": "", // Trading pair
    "timeInForce": "", // Time in force
    "triggerPriceType": "" // Trigger price type
  },
  "returnCode": 0
}
```
