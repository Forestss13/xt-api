#### **<span id="copyTradeUserStatus">跟单用户状态</span>**

| 用户状态 | 描述         |
| ----------- | ------------------- |
| LEADER      | 跟单交易员   |
| FOLLOWER    | 跟单跟随者 |
| NONE        | 无                |
