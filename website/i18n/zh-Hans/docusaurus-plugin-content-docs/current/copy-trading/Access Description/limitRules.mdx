部分接口会有流量控制限制（相应接口会有限流说明）。流量限制主要分为**网关流量限制**和 **WAF 流量限制**。

如果接口请求触发网关流量限制，将返回 **429**，表示访问频率超过限制，**IP** 或 **apiKey** 将被封锁。

### 网关流量限制类型

- **IP 流量限制** 示例：`100/s/ip` → 该接口**每个 IP 每秒**的请求数量限制。

- **apiKey 流量限制** 示例：`50/s/apiKey` → 该接口**每个 apiKey 每秒**的请求数量限制。
