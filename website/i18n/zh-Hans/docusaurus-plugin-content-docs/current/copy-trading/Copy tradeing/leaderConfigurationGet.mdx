**请求类型:** GET

**接口路径:** `/v1/copy-trade-account/public/copy-trade/leader-detail`

### 限流规则

2次/秒/IP

### 请求参数

| 参数名      | 类型   | 是否必填 | 默认值 | 描述       | 取值范围 |
| --------- | ------ | --------- | ------- | ----------------- | ------ |
| accountId | number | 是       |         | 带单账户ID |        |

### 响应示例

```json title="响应"
{
  "ma": [{}],
  "mc": "string",
  "rc": 0,
  "result": {
    "avatar": "string", // 头像
    "currentFollowNumber": 0, // 当前跟单人数
    "displayEquity": true, // 是否展示总资产
    "displayQuantity": true, // 是否展示买入数量
    "fansNum": 0, // 粉丝数
    "followNumber": 0, // 关注数
    "label": [
      // 标签
      "string"
    ],
    "leadDays": 0, // 成为带单者的天数
    "leaderIntro": "string", // 带单者简介
    "level": 0, // 当前等级
    "levelId": 0, // 当前等级ID
    "levelName": "string", // 当前等级名称
    "maxFollowerSize": 0, // 最大跟单人数
    "nickName": "string", // 昵称
    "profitRate": 0, // 分润比例
    "totalRights": 0 // 总资产
  }
}
```
