**请求类型:** GET

**接口路径:** `/v1/copy-trade-balance/public/copy-trade/leader-income-chart`

### 限流规则

2次/秒/IP

### 请求参数

| 参数名 | 类型 | 是否必填 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| `accountId` | number | 是 |  | 带单账户ID |
| `days` | number | 否 | 7 | 查询天数。可选值: `7, 30, 90, 180` |

### 响应示例

```json title="响应"
{
  "ma": [{}],
  "mc": "string",
  "rc": 0,
  "result": [
    {
      "amount": 0, // 每日收益金额
      "id": "string", // ID
      "time": 0 // 时间戳
    }
  ]
}
```
