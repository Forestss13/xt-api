**请求类型:** POST

**接口路径:** `/v1/copy-trade-order/copy-trade/order/close`

### 限流规则

50次/秒/API密钥

### 请求参数

| 参数名          | 类型   | 是否必填 | 默认值 | 描述         | 取值范围 |
| ------------- | ------ | --------- | ------- | --------------- | ------ |
| leaderOrderId | number | 是      |         | 带单订单ID |        |
| symbol        | string | 是      |         | 交易对          |        |
