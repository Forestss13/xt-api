**请求类型:** GET

**接口路径:** `/v1/copy-trade-market/public/copy-trade/symbol`

### 限流规则

2次/秒/API密钥

### 请求参数

_无_

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": [
    {
      "symbolId": 1, // 交易对ID
      "symbol": "btc_usdt" // 交易对
    }
  ]
}
```
