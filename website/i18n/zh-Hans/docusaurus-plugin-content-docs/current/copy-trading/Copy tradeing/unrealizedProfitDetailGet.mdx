**类型:** GET

**描述:** `/v1/copy-trade-balance/copy-trade/share-profit/expected-detail`

#### 限流规则

2/s/ip

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| direction | string | false | NEXT | 查询方向 | PREV - 上一页, NEXT - 下一页 |
| limit | number | false | 10 | 页数 | 1-100 |
| id | object | false | - | 起始ID，例如 `6216559590087220004` | - |

### 响应示例

```json title="响应"
{
  "ma": [{}],
  "mc": "string",
  "rc": 0,
  "result": {
    "hasNext": true,
    "hasPrev": true,
    "items": [
      {
        "amount": 0, // 利润分成金额
        "avatar": "string", // 跟单者头像
        "id": 0, // 用于分页查询的id
        "name": "string" // 跟单者昵称
      }
    ]
  }
}
```
