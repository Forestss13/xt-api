**类型:** GET

**描述:** `/v1/copy-trade-order/copy-trade/order/stop-profit-loss`

#### 限流规则

2/s/apikey

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| symbol | string | true | - | 交易对符号 |
| leaderOrderId | number | true | - | 对于跟单订单，需要 `leaderOrderId`；对于带单订单，需要 `orderId` |
| triggerProfitPrice | number | false | - | 触发止盈价格 |
| triggerStopPrice | number | false | - | 触发止损价格 |

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": true
}
```
