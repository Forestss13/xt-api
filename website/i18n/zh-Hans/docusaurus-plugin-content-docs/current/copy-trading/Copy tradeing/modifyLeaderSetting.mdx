**请求类型:** POST

**接口路径:** `/v1/copy-trade-account/copy-trade/leader-settings`

#### 限流规则

2次/秒/IP

### 请求参数

| 参数名 | 类型 | 是否必填 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| `leaderIntro` | string | 否 |  | 带单者简介 |
| `labelIds` | string | 否 |  | 标签ID，逗号分隔，例如 `1,2,3,4` |
| `displayEquity` | bool | 否 |  | 是否展示总资产 |
| `displayQuantity` | bool | 否 |  | 是否展示买入数量 |
| `symbols` | string | 是 |  | 选择的跟单交易对，例如 `btc_usdt,eth_usdt` |
| `openLeader` | bool | 否 |  | 是否开启带单 |

### 响应示例

```json title="响应"
{
  "ma": [{}],
  "mc": "string",
  "rc": 0,
  "result": true
}
```
