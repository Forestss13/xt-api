| 状态码                      | 错误信息               |
| -------------------------- | --------------------- |
| 200                        | 成功                  |
| 401                        | 需要登录              |
| 403                        | 登录已过期            |
| invalid_params             | 参数错误              |
| invalid_order              | 订单非法              |
| invalid_symbol             | 交易对非法            |
| invalid_price              | 价格非法              |
| invalid_order_type         | 订单类型必填          |
| invalid_order_side         | 订单方向已过期        |
| invalid_triggerProfitPrice | 止盈触发价格非法      |
| invalid_triggerStopPrice   | 止损触发价格非法      |
| invalid_leader             | 交易员不存在          |
| invalid_accountId          | 账户 ID 不存在        |
| invalid_leverage           | 杠杆倍数非法          |
