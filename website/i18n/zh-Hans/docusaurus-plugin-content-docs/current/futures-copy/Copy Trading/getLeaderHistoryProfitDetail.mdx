**类型:** GET

**描述:** `/future/copytrade/user/v1/copy-trade/my-profit-history-detail`

### 参数

| 名称       | 类型   | 必填 | 默认值 | 描述     | 范围       |
| ---------- | ------ | ---- | ------ | -------- | ---------- |
| divideTime | number | 是   | -      | 时间     | -          |
| direction  | string | 否   | -      | 分页方向 | NEXT, PREV |
| limit      | number | 否   | -      | 页面大小 | 10         |
| id         | number | 否   | -      | ID       | -          |

### 限流规则

2/s/apikey

### 响应示例

```json
{
  "returnCode": 0,
  "msgInfo": "success",
  "error": null,
  "result": {
    "hasPrev": false,
    "hasNext": false,
    "items": [
      {
        "avatar": "string",   // 头像
        "name": "string",     // 带单员昵称
        "profit": 5,          // 盈利金额
        "profitTime": "time", // 盈利时间
        "coin": 0,            // 币种
        "id":                 // ID
      }
    ]
  }
}
```
