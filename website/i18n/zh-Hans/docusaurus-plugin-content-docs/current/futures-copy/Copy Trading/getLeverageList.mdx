**类型:** GET

**描述:** `/future/copytrade/order/v1/position/list`

### 参数

| 名称                  | 类型   | 必填 | 默认值 | 描述   | 范围 |
| --------------------- | ------ | ---- | ------ | ------ | ---- |
| symbol                | string | 否   | -      | 交易对 | -    |
| welfareAccountVersion | string | 否   | -      | 版本   | -    |

### 限流规则

2/s/apikey

### 响应示例

```json
{
  "returnCode": 0,
  "msgInfo": "success",
  "error": null,
  "result": [
    {
      "symbol": "string", // 交易对
      "positionType": "string", // 仓位类型: CROSSED, ISOLATED
      "positionSide": "string", // 持仓方向
      "contractType": "string", // 合约类型: PERPETUAL, PREDICT
      "positionSize": 0, // 持仓数量
      "closeOrderSize": 0, // 平仓订单数量
      "availableCloseSize": 0, // 可平仓数量
      "entryPrice": 0, // 持仓均价
      "openOrderSize": 0, // 开仓订单数量
      "isolatedMargin": 0, // 逐仓保证金
      "openOrderMarginFrozen": 0, // 开仓订单冻结保证金
      "realizedProfit": 0, // 已实现盈亏
      "autoMargin": false, // 是否自动追加保证金
      "leverage": 0, // 杠杆倍数
      "profitId": 0, // 止盈止损ID
      "triggerPriceType": 0, // 触发价格类型: 1 指数价格, 2 标记价格, 3 成交价格
      "triggerProfitPrice": 0, // 止盈触发价格
      "triggerStopPrice": 0, // 止损触发价格
      "welfareAccount": false // 是否为福利账户
    }
  ]
}
```
