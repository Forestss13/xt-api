**类型:** GET

**描述:** `/future/copytrade/user/v1/copy-trade/follower-position`

### 参数

| 名称            | 类型   | 必填 | 默认值 | 描述         | 范围 |
| --------------- | ------ | ---- | ------ | ------------ | ---- |
| leaderAccountId | number | 是   | -      | 带单员账户ID | -    |

### 限流规则

2/s/apikey
