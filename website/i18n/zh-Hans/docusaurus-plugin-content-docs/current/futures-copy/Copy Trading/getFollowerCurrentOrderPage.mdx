**类型:** GET

**描述:** `/future/copytrade/order/v1/copy-trade/follower-order-page`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| type | number | 否 | - | 类型: 1 (详情), 2 (汇总) | 1, 2 |
| symbol | string | 否 | - | 交易对 | - |
| direction | string | 否 | - | 分页方向 | NEXT, PREV |
| limit | number | 否 | - | 页面大小 | 10 |
| id | number | 否 | - | 记录ID | - |

### 限流规则

2/s/apikey
