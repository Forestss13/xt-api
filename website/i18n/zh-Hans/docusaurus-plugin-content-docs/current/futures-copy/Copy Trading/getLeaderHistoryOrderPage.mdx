**类型:** GET

**描述:** `/future/copytrade/order/v1/copy-trade/leader-order-history`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| type | number | 是 | - | 类型，1 (详情), 2 (汇总) | 1, 2 |
| symbol | string | 否 | - | 交易对 | - |
| direction | string | 否 | - | 分页方向 | NEXT, PREV |
| limit | number | 否 | - | 页面大小 | 10 |
| id | number | 否 | - | ID | - |

### 限流规则

2/s/apikey
