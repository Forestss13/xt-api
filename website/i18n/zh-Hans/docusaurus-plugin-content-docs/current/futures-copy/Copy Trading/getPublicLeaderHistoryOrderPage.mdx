### 接口

`GET /future/copytrade/public/order/v1/public/copy-trade/leader-order-history`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| type | number | 否 | - | 类型 (1:详情, 2:汇总) | 1, 2 |
| symbol | string | 否 | - | 交易对 | - |
| leaderAccountId | number | 是 | - | 带单员账户ID | - |
| direction | string | 否 | - | 查询方向 | NEXT, PREV |
| limit | number | 否 | - | 页面大小 | 10 |
| id | number | 否 | - | 页面ID | - |

### 限流规则

2/s/ip
