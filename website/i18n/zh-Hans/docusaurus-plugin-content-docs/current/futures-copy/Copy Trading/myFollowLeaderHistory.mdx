**类型:** GET

**描述:** `/future/copytrade/user/v1/copy-trade/follower-leaders-page`

---

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| type | number | 否 | - | 类型 (1: 当前跟单, 2: 历史) | 1, 2 |
| direction | string | 否 | - | 方向 | NEXT, PREV |
| limit | number | 否 | - | 页面大小 | 10 |
| id | number | 否 | - | ID | - |

---

### 限流规则

2/s/apikey

---

### 响应示例

```json
{
  "returnCode": 0,
  "msgInfo": "success",
  "error": null,
  "result": {
    "hasPrev": false,
    "hasNext": false,
    "items": [
      {
        "leaderName": "string",     // 带单员昵称
        "avatar": "string",         // 头像
        "accountId": 0,             // 带单员账户ID
        "id": 0,                    // ID
        "level": 0,                 // 等级
        "levelName": "string",      // 等级名称
        "followMarginU": 0,         // 总跟单保证金
        "followProfitU": 0,         // 总跟单收益
        "cancel": Boolean           // 状态: false=跟单中, true=已取消
      }
    ]
  }
}
```
