**类型:** GET

**描述:** `/future/copytrade/public/user/v1/public/copy-trade/leader-detail-v2`

### 参数

| 名称      | 类型   | 必填 | 默认值 | 描述      | 范围 |
| --------- | ------ | --------- | ------- | ---------------- | ------ |
| accountId | number | 是       | -       | 带单员账户ID | -      |

### 限流规则

2/s/ip

### 响应示例

```json
{
  "returnCode": 0,
  "msgInfo": "success",
  "error": null,
  "result": {
    "nickName": "string", // 昵称
    "avatar": "string", // 头像
    "intro": "string", // 简介
    "leadDays": 0, // 带单天数（开始时间）
    "levelId": 0, // 带单员等级ID
    "level": 0, // 带单员等级
    "levelName": "string", // 带单员等级名称
    "label": ["string"], // 标签
    "followNumber": 0, // 累计跟单人数
    "currentFollowNumber": 0, // 当前跟单人数
    "maxFollowerSize": 0, // 最大允许跟单人数
    "profitRate": 0, // 带单员收益率
    "platformProfitRate": 0, // 平台收益率
    "totalRights": 0, // 总权益
    "displayEquity": true, // 是否显示权益
    "displayPosition": false // 是否显示仓位
  }
}
```
