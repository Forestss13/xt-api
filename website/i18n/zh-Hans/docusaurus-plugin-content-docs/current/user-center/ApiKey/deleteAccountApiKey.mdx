---

**类型：** DELETE

**描述：**`/v4/user/account/api-key/{apikeyId}`

### 参数

| 名称     | 类型   | 必填 | 默认值 | 描述 | 取值范围 |
| -------- | ------ | --------- | ------- | ----------- | ------ |
| apikeyId | string | true      | -       | API 密钥 ID   | -      |

---

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [{}],
  "result": true // true: 成功；false/null: 失败
}
```
