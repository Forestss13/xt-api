**类型：** PUT

**描述：** `/v4/user/account/api-key`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| id | string | true | - | API 密钥 ID | - |
| accessKey | string | true | - | 访问密钥 | - |
| keyName | string | false | - | API 密钥名称 | - |
| bindIps | string | false | - | 绑定 IP 列表，多个 IP 用逗号分隔 | - |
| roleScopes | string | true | - | 权限代码 | QUERY: 查询；TRADE: 现货和杠杆交易；WITHDRAW: 提现；LEVER_LOAN: 杠杆质押；FUTURE: 合约交易；TRANSFER: 划转 |
| tags | string | false | - | 标签 | - |

---
