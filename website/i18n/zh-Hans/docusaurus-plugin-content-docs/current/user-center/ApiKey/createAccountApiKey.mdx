**类型：** POST

**描述：** `/v4/user/account/api-key`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| userAccountId | string | true | - | 账户 ID | - |
| keyName | string | true | - | API 密钥名称 | - |
| bindIps | string | false | - | 绑定 IP 列表，多个 IP 用逗号分隔 | - |
| roleScopes | string | true | - | 权限代码 | QUERY: 启用查询权限；TRADE: 启用现货和杠杆交易权限；WITHDRAW: 启用提现权限；LEVER_LOAN: 启用杠杆质押权限；FUTURE: 启用合约交易权限；TRANSFER: 启用划转权限 |
| tags | string | false | - | 标签 | - |

---
