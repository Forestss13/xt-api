**Type:** GET

**Description:** `/v4/user/account/api-key`

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| userAccountId | string | false | - | Account id | - |
| keyName | string | false | - | ApiKey name | - |
| isLock | integer | false | - | ApiKey is locked or not | 0: not locked; 1: locked |

---
