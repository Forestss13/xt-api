**类型:** GET

**描述:** `/v4/user/account/vip-info`

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "vipLevel": 0, // VIP 等级
    "vipName": "VIP0", // VIP 名称
    "makerFeeRate": 0.002, // Maker 费率
    "takerFeeRate": 0.001 // Taker 费率
  }
}
```
