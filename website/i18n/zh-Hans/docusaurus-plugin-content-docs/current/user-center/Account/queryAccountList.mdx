**类型:** GET

**描述:** `/v4/user/account`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| accountId | string | 否 | - | 账户 ID | - |
| accountName | string | 否 | - | 账户名称 | - |
| level | integer | 否 | - | 账户级别 | 1: 主账户; 2: 子账户 |

---
