**类型:** GET

**描述:** `/v4/lever/loan`

### 参数

| 名称      | 类型   | 必需 | 默认值 | 描述    | 范围      |
| --------- | ------ | --------- | ------- | -------------- | ----------- |
| symbol    | 字符串 | Yes       | -       | Trading pair   | -           |
| startTime | long   | Yes       | -       | Start time     | -           |
| endTime   | long   | Yes       | -       | End time       | -           |
| fromId    | long   | No        | -       | Page start ID  | -           |
| direction | 字符串 | Yes       | -       | Page direction | NEXT / PREV |
| limit     | int    | No        | 20      | Page limit     | 默认值 20  |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [{}],
  "result": {
    "has下一页": 是, // whether has next page
    "hasPrev": 是, // whether has previous page
    "items": [
      {
        "id": "字符串", // loan record id
        "amount": 0, // loan amount
        "currency": "字符串", // currency
        "currencyId": 0, // 货币ID
        "symbol": "字符串", // trading pair symbol
        "symbolId": 0, // symbol id
        "createTime": 0 // create time (timestamp)
      }
    ]
  }
}
```
