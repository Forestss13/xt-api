**类型：** GET

**描述：** `/v4/lever/loan`

### 参数

| 名称      | 类型   | 必需 | 默认值 | 描述        | 范围        |
| --------- | ------ | ---- | ------ | ----------- | ----------- |
| symbol    | string | 是   | -      | 交易对      | -           |
| startTime | long   | 是   | -      | 开始时间    | -           |
| endTime   | long   | 是   | -      | 结束时间    | -           |
| fromId    | long   | 否   | -      | 分页起始 ID | -           |
| direction | string | 是   | -      | 分页方向    | NEXT / PREV |
| limit     | int    | 否   | 20     | 分页限制    | 默认 20     |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [{}],
  "result": {
    "hasNext": true, // 是否有下一页
    "hasPrev": true, // 是否有上一页
    "items": [
      {
        "id": "string", // 借款记录 ID
        "amount": 0, // 借款金额
        "currency": "string", // 币种
        "currencyId": 0, // 币种 ID
        "symbol": "string", // 交易对
        "symbolId": 0, // 交易对 ID
        "createTime": 0 // 创建时间（时间戳）
      }
    ]
  }
}
```
