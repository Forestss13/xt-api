**类型:** GET

**描述:** `/v4/lever/repay`

### 参数

| 名称      | 类型   | 必需 | 默认值 | 描述    | 范围      |
| --------- | ------ | --------- | ------- | -------------- | ----------- |
| Symbol    | String | No        | -       | Trading pair   | -           |
| StartTime | Long   | Yes       | -       | Start time     | -           |
| EndTime   | Long   | No        | -       | End time       | -           |
| FromId    | Long   | No        | -       | Page start ID  | -           |
| Direction | String | Yes       | -       | Page direction | NEXT / PREV |
| Limit     | Int    | No        | 20      | Page limit     | 默认值 20  |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [{}],
  "result": {
    "has下一页": 是, // Whether has next page
    "hasPrev": 是, // Whether has previous page
    "items": [
      {
        "id": "字符串", // Repay record ID
        "amount": 0, // Repay amount
        "currency": "字符串", // 货币
        "currencyId": 0, // 货币 ID
        "symbol": "字符串", // Trading pair symbol
        "symbolId": 0, // Symbol ID
        "interestAmount": 0, // Interest amount
        "type": "MANUAL", // Repay type: MANUAL / SYSTEM
        "createTime": 0 // Create time (timestamp)
      }
    ]
  }
}
```
