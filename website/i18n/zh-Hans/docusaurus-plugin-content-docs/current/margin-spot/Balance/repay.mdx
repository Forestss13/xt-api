**类型：** GET

**描述：** `/v4/lever/repay`

### 参数

| 名称      | 类型   | 必需 | 默认值 | 描述        | 范围        |
| --------- | ------ | ---- | ------ | ----------- | ----------- |
| Symbol    | String | 否   | -      | 交易对      | -           |
| StartTime | Long   | 是   | -      | 开始时间    | -           |
| EndTime   | Long   | 否   | -      | 结束时间    | -           |
| FromId    | Long   | 否   | -      | 分页起始 ID | -           |
| Direction | String | 是   | -      | 分页方向    | NEXT / PREV |
| Limit     | Int    | 否   | 20     | 分页限制    | 默认 20     |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [{}],
  "result": {
    "hasNext": true, // 是否有下一页
    "hasPrev": true, // 是否有上一页
    "items": [
      {
        "id": "string", // 还款记录 ID
        "amount": 0, // 还款金额
        "currency": "string", // 币种
        "currencyId": 0, // 币种 ID
        "symbol": "string", // 交易对
        "symbolId": 0, // 交易对 ID
        "interestAmount": 0, // 利息金额
        "type": "MANUAL", // 还款类型：MANUAL / SYSTEM
        "createTime": 0 // 创建时间（时间戳）
      }
    ]
  }
}
```
