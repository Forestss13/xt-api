**类型:** GET

**描述:** `/v4/lever/balance`

### 参数

| 名称   | 类型   | 必需 | 默认值 | 描述             | 范围 |
| ------ | ------ | --------- | ------- | ----------------------- | ------ |
| symbol | 字符串 | 是      | -       | Symbol, e.g. `BTC_USDT` | -      |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [{}],
  "result": {
    "btcLoanAmount": 0, // convert BTC loan assets
    "btcNetAmount": 0, // convert BTC net assets
    "symbol": "字符串", // symbol
    "symbolId": 0, // symbol id
    "usdtLoanAmount": 0, // convert USDT loan assets
    "usdtNetAmount": 0, // convert USDT net assets
    "base": {
      "availableAmount": "字符串", // available amount
      "capitalAmount": "字符串", // capital amount
      "currency": "字符串", // currency
      "currencyId": 0, // 货币ID
      "frozenAmount": "字符串", // frozen amount
      "interestAmount": "字符串", // interest amount outstanding
      "loanAmount": "字符串", // loan amount
      "totalAmount": "字符串", // total amount
      "updatedTime": 0 // update time
    },
    "quote": {
      "availableAmount": "字符串", // available amount
      "capitalAmount": "字符串", // capital amount
      "currency": "字符串", // currency
      "currencyId": 0, // 货币ID
      "frozenAmount": "字符串", // frozen amount
      "interestAmount": "字符串", // interest amount outstanding
      "loanAmount": "字符串", // loan amount
      "totalAmount": "字符串", // total amount
      "updatedTime": 0 // update time
    }
  }
}
```
