### 注意事项

Due to reasons such as high latency and poor stability, it 不推荐 to access the API through a proxy.

- GET request parameters should be placed in **查询参数 params**.
- POST request parameters should be placed in the **request 请求体**.
- 请设置请求头信息为：Content-类型=application/json

For requests that start other than /public, the request message needs to be signed
