### HTTP 状态码

| httpStatus | 描述 |
| --- | --- |
| 200 | 请求成功，请进一步检查 rc 和 mc 部分 |
| 404 | 接口不存在 |
| 429 | 请求过于频繁，请根据速率限制要求控制请求频率 |
| 500 | 服务异常 |
| 502 | 网关异常 |
| 503 | 服务不可用，请稍后重试 |

### 返回代码 (rc)

| rc  | 返回代码         |
| --- | ---------------- |
| 0   | 业务成功         |
| 1   | 业务失败         |
