### 注意事项

由于延迟高、稳定性差等原因，不建议通过代理访问 API。

- GET 请求参数应放在 **query params** 中。
- POST 请求参数应放在 **request body** 中。
- 请将请求头信息设置为：Content-Type=application/json

对于非 /public 开头的请求，请求消息需要进行签名
