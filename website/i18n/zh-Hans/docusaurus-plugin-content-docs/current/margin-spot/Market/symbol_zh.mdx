**类型：** GET

**描述：** `/v4/public/lever/symbol`

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": [
    {
      "symbolId": 123, // 现货交易对 ID
      "symbol": "btc_usdt", // 现货交易对
      "buyCurrencyId": 123, // 买入币种 ID
      "buyCurrency": "btc", // 买入币种
      "sellCurrencyId": 234, // 卖出币种 ID
      "sellCurrency": "usdt", // 卖出币种
      "maxLoanAmountBuy": "", // 可借买入币种的最大金额
      "maxLoanAmountSell": "", // 可借卖出币种的最大金额
      "maxLeverage": "", // 最大借款倍数
      "dailyInterestRate": "", // 日利率
      "liquidationRate": "", // 强平风险率
      "tipsRate": "" // 提醒系数
    }
  ]
}
```
