**类型:** POST

**描述:** `/v4/lever/order`

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| Market名称 | String | Yes | - | Trading pair, e.g. `BTC_USDT` | - |
| Coin名称 | String | Yes | - | Coin name, lowercase. e.g. `btc` | - |
| Amount | Number | Yes | - | Amount, e.g. `8000` | - |
| 类型 | String | Yes | - | [订单 类型](#margin订单类型), supports: `LOAN` / `REPAY` | - |
| Media | String | No | - | Media source, e.g. `btok` | - |
| MediaChannel | String | No | - | Media channel | - |

#### margin订单类型

| Value | 描述 |
| --- | --- |
| LOAN | 借款 |
| REPAY | 还款 |
