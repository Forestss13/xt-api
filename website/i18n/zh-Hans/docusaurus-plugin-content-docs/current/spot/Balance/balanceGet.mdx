**类型：** GET

**描述：** `/v4/balance`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| currency | string | true | N/A | 币种名称（例如：usdt、btc） |  |

### 注意事项

限流规则：**3次/秒/用户**

此接口用于获取现货账户中特定币种的余额信息。

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {
    "currency": "usdt",
    "currencyId": 11,
    "frozenAmount": 0,
    "availableAmount": 1000.5,
    "totalAmount": 1000.5,
    "convertBtcAmount": 0.03
  }
}
```
