**类型：** GET

**描述：** `/v4/public/currencies`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述           | 范围 |
| ---- | ---- | -------- | ------ | -------------- | ---- |
| –    | –    | false    | N/A    | 无需任何参数   | –    |

### 注意事项

此接口用于获取所有支持币种的信息。

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": [
    {
      "id": 11, // 币种 id
      "currency": "usdt", // 币种名称
      "fullName": "usdt", // 币种全称
      "logo": null, // 币种 logo
      "cmcLink": null, // cmc 链接
      "weight": 100,
      "maxPrecision": 6,
      "depositStatus": 1, // 充值状态（0=关闭，1=开启）
      "withdrawStatus": 1, // 提现状态（0=关闭，1=开启）
      "convertEnabled": 1, // 小额资产兑换开关【0=关闭，1=开启】
      "transferEnabled": 1 // 划转开关【0=关闭，1=开启】
    }
  ]
}
```
