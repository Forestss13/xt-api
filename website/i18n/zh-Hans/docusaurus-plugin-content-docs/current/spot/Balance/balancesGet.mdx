**类型:** GET

**描述:** `/v4/balances`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| currencies | string | false | N/A | 货币列表，逗号分隔 (例如: usdt,btc) |  |

### 注意事项

限流规则: **10/s/apikey**

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {
    "totalBtcAmount": 0,
    "assets": [
      {
        "currency": "string",
        "currencyId": 0,
        "frozenAmount": 0,
        "availableAmount": 0,
        "totalAmount": 0,
        "convertBtcAmount": 0
      }
    ]
  }
}
```
