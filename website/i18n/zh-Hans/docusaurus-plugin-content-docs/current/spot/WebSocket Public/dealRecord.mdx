### 请求

- **格式:** `trade@{symbol}`
- **示例:** `trade@btc_usdt`
- **频率:** 实时

### 响应示例

```json title="响应"
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "i": 6316559590087222000,
    "t": 1655992403617,
    "oi": 6616559590087222666,
    "p": "43000",
    "q": "0.21",
    "v": "9030",

  }
}
```
