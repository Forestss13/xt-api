### 请求

- **格式:** `depth@{symbol},{levels}`
- **档位:** 5, 10, 20, 50
- **示例:** `depth@btc_usdt,20`
- **频率:** 1000ms

### 响应示例

```json title="响应"
{
  "topic": "depth",
  "event": "depth@btc_usdt,20",
  "data": {
    "s": "btc_usdt",
    "i": 1657699200000,
    "a": [
      ["34000", "1.2"],
      ["34001", "2.3"]
    ],
    "b": [
      ["32000", "0.2"],
      ["31000", "0.5"]
    ]
  }
}
```
