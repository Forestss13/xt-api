### 请求

- **格式:** `depth_update@{symbol}`
- **示例:** `depth_update@btc_usdt`
- **频率:** 100ms

### 响应示例

```json title="响应"
{
  "topic": "depth_update",
  "event": "depth_update@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "fi": 121,
    "i": 123,
    "a": [
      ["34000", "1.2"],
      ["34001", "2.3"]
    ],
    "b": [
      ["32000", "0.2"],
      ["31000", "0.5"]
    ]
  }
}
```
