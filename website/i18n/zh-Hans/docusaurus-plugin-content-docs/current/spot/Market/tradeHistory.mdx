**类型:** GET **描述:** `/v4/public/trade/history`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 是 |  | trading pair |  |
| limit | 数字 | 否 | 200 |  | 1~1000 |
| direction | 字符串 | 是 |  | 查询参数 direction | PREV - previous page, NEXT - next page |
| fromId | 数字 | 否 |  | Start ID, e.g. 6216559590087220004 |  |

#### **Limit Flow Rules**

10/s/ip
