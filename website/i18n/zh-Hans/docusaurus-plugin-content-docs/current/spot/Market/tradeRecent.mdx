**类型:** GET  
**描述:** `/v4/public/trade/recent`

### 参数

| 名称   | 类型   | 是否必需 | 默认值 | 描述   | 取值范围 |
| ------ | ------ | -------- | ------ | ------ | -------- |
| symbol | string | true     |        | 交易对 |          |
| limit  | number | false    | 200    |        | 1~1000   |

#### **限流规则**

10次/秒/IP

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": [
    {
      "i": 0,
      "t": 0,
      "p": "string",
      "q": "string",
      "v": "string",
      "b": true
    }
  ]
}
```
