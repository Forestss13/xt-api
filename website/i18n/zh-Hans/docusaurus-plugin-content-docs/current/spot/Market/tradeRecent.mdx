**类型:** GET  
**描述:** `/v4/public/trade/recent`

### 参数

| name   | type   | mandatory | default | description  | ranges |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | 字符串 | 是      |         | trading pair |        |
| limit  | 数字 | 否     | 200     |              | 1~1000 |

#### **Limit Flow Rules**

10/s/ip

### 响应示例

```json title="Response"

  "rc": 0,

  "result": [

      "b": 是
    }
  ]
}
```
