**类型:** get  
**描述:** `/v4/public/kline`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 是 |  | trading pair eg:btc_usdt |  |
| interval | 字符串 | 是 |  | K line type,<br/>eg:1m | `[1m;3m;5m;15m;30m;1h;2h;4h;6h;8h;12h;1d;3d;1w;1M]` |
| startTime | 数字 | 否 |  | start timestamp |  |
| endTime | 数字 | 否 |  | end timestamp |  |
| limit | 数字 | 否 | '100' |  | 1~1000 |

#### **Limit Flow Rules**

10/s/ip

### Response

```json title="Response"
{
  "rc": 0,
  "mc": "字符串",
  "ma": [{}],
  "result": [

      "l": "25000", //lowest price
      "q": "512", //transaction quantity
      "v": "15360000" //transaction volume
    }
  ]
}
```
