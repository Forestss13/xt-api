**类型:** GET **描述:** `/v4/public/ticker/book`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | false |  | 交易对 例如：btc_usdt |  |
| symbols | array | false |  | 交易对集合，优先级高于 symbol。例如：btc_usdt,eth_usdt |  |
| tags | string | false |  | 标签集合，用逗号分隔，目前仅支持 spot |  |

#### **限流规则**

1. 单个交易对：`10次/秒/IP`
2. 多个交易对：`10次/秒/IP`

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": [
    {
      "s": "btc_usdt",
      "t": 1661856036925,

      "bq": null
    }
  ]
}
```
