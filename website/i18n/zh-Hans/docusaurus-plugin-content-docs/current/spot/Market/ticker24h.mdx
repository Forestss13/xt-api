**类型:** GET **描述:** `/v4/public/ticker/24h`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 否 |  | trading pair eg:btc_usdt |  |
| symbols | 数组 | 否 |  | Collection of trading pairs. Priority is higher than symbol. eg: btc_usdt,eth_usdt |  |
| tags | 字符串 | 否 |  | Set of tags, separated by commas, currently only supports spot |  |

#### **Limit Flow Rules**

1. single symbol: `10/s/ip`
2. multiple symbols: `10/s/ip`
