**类型:** GET **描述:** `/v4/public/depth`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true |  | 交易对 例如：btc_usdt |  |
| limit | number | false | 100 | 最小查询数量为 100 | 1~500 |

#### **限流规则**

10次/秒/IP

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "timestamp": 1662445330524,
    "lastUpdateId": 137333589606963580,
    "bids": [
      ["200.0000", "0.996000"],
      ["100.0000", "0.001000"],
      ["20.0000", "10.000000"]
    ],
    "asks": []
  }
}
```
