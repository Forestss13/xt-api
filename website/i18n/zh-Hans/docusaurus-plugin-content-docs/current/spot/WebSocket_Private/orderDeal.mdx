**类型:** Websocket Push **描述:** 交易（订单成交）通知

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| – | – | false | N/A | 无需参数, 订阅 **trade** 主题以接收更新 |

### 注意事项

- 当交易（订单成交）发生时触发此推送通知。
- 订阅格式：`trade`
- 指示执行详情，包括交易ID、订单ID、价格、数量和maker/taker信息。
- 用于实时跟踪已完成的交易。

### Push 示例

```json title="Push"
{
  "topic": "trade",
  "event": "trade",
  "data": {
    "s": "btc_usdt", // 交易对
    "i": 6316559590087222000, // 交易ID
    "t": 1655992403617, // 交易时间（毫秒）
    "oi": 6616559590087222666, // 订单ID
    "p": "43000", // 价格
    "q": "0.21", // 数量
    "v": "9030", // 计价数量
    "b": true, // 买方是否为maker（true=买方为maker）
    "tm": 1 // Taker/maker：1 = taker，2 = maker
  }
}
```
