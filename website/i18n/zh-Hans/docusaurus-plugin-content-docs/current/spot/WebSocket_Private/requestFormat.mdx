**类型:** Websocket Info **描述:** WebSocket请求消息的标准格式

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述            |
| ---- | ---- | --------- | ------- | ---------------------- |
| –    | –    | false     | N/A     | 无需参数 |

### 注意事项

- 请求消息用于**订阅**或**取消订阅**WebSocket主题。
- 参数格式：\{topic\}@\{arg\},\{arg\},…
- `listenKey` 必须通过 **/v4/ws-token** 接口申请。
- `id` 是用于匹配响应的自定义请求标识符。

### 订阅示例

```javascript title="Subscribe"
{
"方法": "subscribe",
"params": [
  "\{topic\}@\{arg\},\{arg\}",   // 事件订阅
  "\{topic\}@\{arg\}"
],
"listenKey": "512312356123123123", // 来自 /v4/ws-token 的监听密钥
"id": "\{id\}"
}
```
