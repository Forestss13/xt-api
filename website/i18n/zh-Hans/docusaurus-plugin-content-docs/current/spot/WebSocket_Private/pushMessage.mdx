### Format

```javascript title="Format (with comments)"
{
  "topic": "trade",          // 频道名称
  "event": "trade@btc_usdt", // 带交易对的事件标识符
  "data": {}                 // 载荷数据
}
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {}
}
示例: 交易记录（实时推送）

{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "i": 6316559590087222000,
    "t": 1655992403617,
    "oi": 6616559590087222666,
    "p": "43000",
    "q": "0.21",
    "v": "9030",
    "b": true
  }
}
```
