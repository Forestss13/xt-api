**类型:** Websocket Push **描述:** 委托（触发订单）变化通知

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| – | – | false | N/A | 无需参数, 订阅 **trigger** 主题以接收更新 |

### 注意事项

- 当**触发/委托订单**发生变化时触发此推送通知。
- 订阅格式：`trigger`
- 用于跟踪触发订单的创建、更新和状态变化。

### Push 示例

```json title="Push"
{
  "topic": "trigger",
  "event": "trigger",
  "data": {
    "s": "btc_usdt", // 交易对
    "t": 1656043204763, // 事件时间（毫秒）
    "i": "6216559590087220004", // 触发ID
    "st": "NEW" // 状态（如：NEW, FILLED, CANCELED）
  }
}
```
