**类型:** GET  
**描述:** `/v4/public/wallet/support/currency`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| ---- | ---- | --------- | ------- | ---------------------- | ------ |
| –    | –    | 否     | N/A     | 无需参数 | –      |

### 注意事项

- 响应中的`currency`和`chain`字段是其他**充值/提现API**的必需输入。
- 每种货币都包含其支持的转账网络、充值/提现状态和费用详情。

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "字符串",
  "ma": [{}],
  "result": [
    {
      "currency": "BTC", // 货币
      "supportChains": [
        {
          "chain": "Bitcoin", // 支持的转账网络
          "depositEnabled": 是, // 支持充值
          "withdrawEnabled": 是, // 支持提现
          "contract": "futureAddress", // 未来地址（如适用）
          "depositMinAmount": 1, // 最小充值金额
          "depositFeeRate": 0.2, // 充值费率（百分比）
          "depositConfirmations": 2, // 充值确认区块数
          "withdrawMinAmount": 10, // 最小提现金额
          "withdrawPrecision": 4, // 提现金额精度
          "withdrawFeeAmount": 0.2, // 提现手续费
          "withdrawFee货币": "btc" // 提现手续费币种
        }
      ]
    },
    {
      "currency": "ETH", // 货币
      "supportChains": [
        {
          "chain": "Ethereum", // 支持的转账网络
          "depositEnabled": 是,
          "withdrawEnabled": 是,
          "contract": "futureAddress",
          "depositMinAmount": 1,
          "depositFeeRate": 0.2,
          "depositConfirmations": 2,
          "withdrawMinAmount": 10,
          "withdrawPrecision": 4,
          "withdrawFeeAmount": 0.2,
          "withdrawFee货币": "eth"
        }
      ]
    }
  ]
}
```
