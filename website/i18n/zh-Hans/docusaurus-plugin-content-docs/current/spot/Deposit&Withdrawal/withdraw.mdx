**类型:** POST **描述:** `/v4/withdraw`

### 参数

> 📌 注意：参数需以 JSON 格式放置在**请求体**中。

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| currency | string | 是 | N/A | 币种名称，来自**获取充提币支持币种**接口 |  |
| chain | string | 否 | N/A | 转账网络名称，来自**获取充提币支持币种**接口 |  |
| clientOrderId | string | 否 | N/A | 自定义客户端ID，正则表达式：`^[a-zA-Z0-9_]{4,32}$` |  |
| amount | number | 是 | N/A | 提币数量（包含手续费） |  |
| address | string | 否 | N/A | 提币地址 |  |
| memo | string | 否 | N/A | Memo/Tag（某些链如EOS必需） |  |
| toAccountId | number | 否 | N/A | 接收用户ID |  |

### 注意事项

- 参数必须以 JSON 格式放置在请求体中。
- 限流规则：**每个apikey每秒1次请求**。
- 对于类EOS链，必须提供 `memo`。
- 返回的 `id` 可用于后续查询提币历史。

### 请求示例

```json title="请求"
{
  "currency": "zb",
  "chain": "Ethereum",
  "amount": 1000,
  "address": "0xfa3abfa50eb2006f5be7831658b17aca240d8526",
  "memo": ""
}
```
