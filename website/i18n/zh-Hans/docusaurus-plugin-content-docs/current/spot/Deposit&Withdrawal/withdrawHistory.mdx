**类型:** GET **描述:** `/v4/withdraw/history`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| currency | 字符串 | 是 | N/A | 货币名称, from **Get supported currencies for deposit/withdrawal** API |  |
| chain | 字符串 | 是 | N/A | Transfer network, from **Get supported currencies for deposit/withdrawal** API |  |
| status | 字符串 | 否 | N/A | Withdrawal status (see [depositWithdrawStatus](#depositWithdrawStatus)) | SUBMIT, REVIEW, AUDITED, AUDITED_AGAIN, PENDING, SUCCESS, FAIL, CANCEL |
| fromId | long | 否 | N/A | The last pagination ID, i.e. the primary key ID of the record |  |
| direction | 字符串 | 否 | NEXT | Pagination direction | NEXT = next page, PREV = previous page |
| limit | int | 否 | 10 | Number of records per page (max 200) | 1 ≤ limit ≤ 200 |
| startTime | long | 否 | N/A | Query range start boundary (timestamp in ms) |  |
| endTime | long | 否 | N/A | Query range end boundary (timestamp in ms) |  |

#### depositWithdrawStatus

| Status | 描述 |
| --- | --- |
| SUBMIT | 已提交 |
| REVIEW | 审核中 |
| AUDITED | 已审核 |
| AUDITED_AGAIN | 重新审核 |
| PENDING | 处理中 |
| SUCCESS | 成功 |
| FAIL | 失败 |
| CANCEL | 已取消 |

### 注意事项

- Supports pagination using `fromId` + `direction`.
- Limit per page is configurable with `limit` (默认值值值值 10, max 200).
- Time range filtering is supported with `startTime` and `endTime`.
- Status values align with **Deposit/Withdrawal record status** definitions.
