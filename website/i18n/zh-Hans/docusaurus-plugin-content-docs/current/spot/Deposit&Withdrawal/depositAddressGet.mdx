**类型:** GET  
**描述:** `/v4/deposit/address`

### 参数

| 名称     | 类型   | 必填 | 默认值值值值 | 描述         | 范围 |
| -------- | ------ | --------- | ------- | ------------------- | ------ |
| chain    | 字符串 | 是      | N/A     | 充值网络 |        |
| currency | 字符串 | 是      | N/A     | 货币名称       |        |

### 注意事项

此接口获取指定货币在给定链上的充值地址。

- 某些货币可能需要在地址之外提供**备注/标签**。
- 始终确认网络与货币匹配，以避免资金损失。

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "字符串",
  "ma": [{}],
  "result": {
    "address": "0xfa3abfa50eb2006f5be7831658b17aca240d8526", // 钱包地址
    "memo": "" // 如需要备注/标签，否则为空
  }
}
```
