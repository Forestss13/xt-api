**类型:** GET **描述:** `/v4/deposit/history`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| currency | 字符串 | 是 | N/A | 货币名称, from **Get supported currencies for deposit/withdrawal** API |  |
| chain | 字符串 | 是 | N/A | Transfer network, from **Get supported currencies for deposit/withdrawal** API |  |
| status | 字符串 | 否 | N/A | Deposit status | SUBMIT, REVIEW, AUDITED, PENDING, SUCCESS, FAIL, CANCEL |
| fromId | long | 否 | N/A | Start ID (e.g. `6216559590087220004`) |  |
| direction | 字符串 | 否 | NEXT | Query direction | PREV, NEXT |
| limit | int | 否 | 10 | Limit 数字, max 200 | 1 ≤ limit ≤ 200 |
| startTime | long | 否 | N/A | Start time for filtering deposit list (timestamp in ms) |  |
| endTime | long | 否 | N/A | End time for filtering deposit list (timestamp in ms) |  |

### 注意事项

This endpoint retrieves the historical deposit records of a user.

- Supports pagination using `fromId` and `direction`.
- Status values reference: [depositWithdrawStatus](#depositWithdrawStatus).
- Some currencies require **memo/tag** to be filled when depositing.
