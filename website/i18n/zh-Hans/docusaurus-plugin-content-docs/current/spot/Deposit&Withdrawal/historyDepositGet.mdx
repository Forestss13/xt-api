**类型:** GET **描述:** `/v4/deposit/history`

### 参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| currency | string | 是 | N/A | 币种名称，来自**获取充提币支持币种**接口 |  |
| chain | string | 是 | N/A | 转账网络，来自**获取充提币支持币种**接口 |  |
| status | string | 否 | N/A | 充币状态 | SUBMIT, REVIEW, AUDITED, PENDING, SUCCESS, FAIL, CANCEL |
| fromId | long | 否 | N/A | 起始ID（例如：`6216559590087220004`） |  |
| direction | string | 否 | NEXT | 查询方向 | PREV, NEXT |
| limit | int | 否 | 10 | 限制数量，最大200 | 1 ≤ limit ≤ 200 |
| startTime | long | 否 | N/A | 筛选充币列表的起始时间（毫秒时间戳） |  |
| endTime | long | 否 | N/A | 筛选充币列表的结束时间（毫秒时间戳） |  |

### 注意事项

此接口用于获取用户的历史充币记录。

- 支持使用 `fromId` 和 `direction` 进行分页。
- 状态值参考：[depositWithdrawStatus](#depositWithdrawStatus)。
- 某些币种充币时需要填写 **memo/tag**。
