### 格式

```javascript title="格式（含注释）"
{
  "topic": "trade",          // 频道名称
  "event": "trade@btc_usdt", // 事件标识符（含交易对）
  "data": {}                 // 数据载荷
}
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {}
}
示例：成交记录（实时推送）
```
