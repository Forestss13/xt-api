**类型:** DELETE **描述:** `/v4/open-order`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 否 |  | Trading pair, if not filled in, represents all |  |
| biz类型 | 字符串 | 是 |  | SPOT, LEVER |  |
| side | 字符串 | 否 |  | BUY, SELL |  |

#### **Limit Flow Rules**

10/s/apikey

**注意:** The parameters should be placed in the request 请求体 in the form of JSON.
