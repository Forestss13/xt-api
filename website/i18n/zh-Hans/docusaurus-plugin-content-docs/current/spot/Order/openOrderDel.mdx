**类型:** DELETE **描述:** `/v4/open-order`

### 参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | 否 |  | 交易对，不填表示所有 |  |
| bizType | string | 是 |  | SPOT（现货）、LEVER（杠杆） |  |
| side | string | 否 |  | BUY（买入）、SELL（卖出） |  |

#### **限流规则**

10次/秒/apikey

**注意:** 参数应以 JSON 形式放在请求体中。

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {}
}
```
