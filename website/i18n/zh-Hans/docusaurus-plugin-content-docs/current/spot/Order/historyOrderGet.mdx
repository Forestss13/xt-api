**类型:** GET **描述:** `/v4/history-order`

### 参数

| 名称 | 类型 | 必填 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 否 |  | Trading pair, if not filled in, represents all |  |
| biz类型 | 字符串 | 否 |  | SPOT, LEVER |  |
| side | 字符串 | 否 |  | BUY, SELL |  |
| 类型 | 字符串 | 否 |  | LIMIT, MARKET |  |
| state | 字符串 | 否 |  | order state: PARTIALLY_FILLED, FILLED, CANCELED, REJECTED, EXPIRED |  |
| fromId | 数字 | 否 |  | start id |  |
| direction | 字符串 | 否 |  | 查询参数 direction: PREV, NEXT |  |
| limit | 数字 | 否 | 20 | Limit 数字, max 100 |  |
| startTime | 数字 | 否 |  | eg: 1657682804112 |  |
| endTime | 数字 | 否 |  |  |  |
| hiddenCanceled | bool | 否 |  |  |  |

#### **Limit Flow Rules**

10/s/apikey
