**类型:** put **描述:** `/v4/order/{orderId}`

### 参数

| 名称     | 类型   | 是否必填 | 默认值 | 描述    | 取值范围 |
| -------- | ------ | -------- | ------ | ------- | -------- |
| orderId  | number | 是       |        | 订单ID  |          |
| price    | number | 是       |        | 价格    |          |
| quantity | number | 是       |        | 数量    |          |

#### **限流规则**

50次/秒/apikey

{

  "ma": [{}],
  "result": {
