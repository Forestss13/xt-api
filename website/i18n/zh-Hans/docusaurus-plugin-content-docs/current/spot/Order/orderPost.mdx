**类型:** post **描述:** `/v4/order`

### 参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | 是 |  | 交易对 |  |
| clientOrderId | string | 否 |  | 客户端订单ID，格式: ^[a-zA-Z0-9_]{4,22}$ |  |
| side | string | 是 |  | 买卖方向：BUY（买入）、SELL（卖出） |  |
| type | string | 是 |  | 订单类型：LIMIT（限价单）、MARKET（市价单） |  |
| timeInForce | string | 是 |  | 有效方式：GTC、FOK、IOC、GTX |  |
| bizType | string | 是 |  | 业务类型：SPOT（现货）、LEVER（杠杆） |  |
| price | number | 否 |  | 价格。限价单必填；市价单不填 |  |
| quantity | number | 否 |  | 数量。限价单必填，或按数量下市价单时必填 |  |
| quoteQty | number | 否 |  | 金额。按金额下市价单时必填 |  |
| nftId | string | 否 |  | NFT ID |  |

#### 备注

创建市价买单时，quantity 必须为空，quoteQty 必填。创建市价卖单时，quoteQty 必须为空，quantity 必填。

#### 限流规则

20次/秒/apikey
