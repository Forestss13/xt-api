**类型:** DELETE **描述:** `/v4/batch-order`

### 参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| clientBatchId | string | 否 |  | 客户端批次ID |  |
| orderIds | array | 是 |  | 订单ID数组，例如：6216559590087220004, 6216559590087220005 |  |

> 注意：参数应以 JSON 形式放在请求体中。

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {}
}
```
