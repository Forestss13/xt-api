**类型:** GET **描述:** `/v4/order`

### 参数

| 名称          | 类型   | 是否必填 | 默认值 | 描述           | 取值范围 |
| ------------- | ------ | -------- | ------ | -------------- | -------- |
| orderId       | number | 否       |        | 订单ID         |          |
| clientOrderId | string | 否       |        | 客户端订单ID   |          |
