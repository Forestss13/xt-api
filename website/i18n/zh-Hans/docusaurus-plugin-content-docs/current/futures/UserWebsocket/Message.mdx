### 订阅 / 取消订阅

```json title="订阅/取消订阅"
{
  "method": "SUBSCRIBE/UNSUBSCRIBE",
  "params": ["{notify}@{listenKey}"],
  "id": "{id}"
}
```

---

### 响应示例

```json title="响应"
{
  "topic": "notify",
  "event": "notify",
  "data": {
    "symbol": "btc_usdt",
    "positionType": "ISOLATED",
    "positionSide": "LONG",
    "positionSize": "123", // 仓位数量
    "notifyType": "WARN" // 通知类型：WARN（警告，即将被强制平仓）、PARTIAL（部分强制平仓）、LIQUIDATION（强制平仓）、ADL（自动减仓）
  }
}
```
