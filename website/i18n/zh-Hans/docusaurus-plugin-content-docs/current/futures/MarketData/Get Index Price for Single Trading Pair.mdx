**类型:** GET

**描述:** `/future/market/v1/public/q/symbol-index-price`

#### 注意：

此方法不需要签名.

### 参数

| 名称   | 类型   | 必需 | 默认值值值值 | 描述  | 范围 |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | 字符串 | 是      | N/A     | Trading pair |        |

### 响应 示例

import CodeBlock from '@theme/CodeBlock';

<CodeBlock language="json" title="响应">
  {`{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "p": 0,  // 价格
    "s": "", // Trading pair
    "t": 0   // Time
  },
  "returnCode": 0
}`}
</CodeBlock>
