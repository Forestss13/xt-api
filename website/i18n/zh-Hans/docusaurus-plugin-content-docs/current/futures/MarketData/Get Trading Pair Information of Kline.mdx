**类型:** GET

**描述:** `/future/market/v1/public/q/kline`

#### 注意：

此方法不需要签名.

### 参数

| 名称 | 类型 | 必需 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 是 | N/A | Trading pair |  |
| interval | 字符串 | 是 | N/A | Time-interval | 1m;5m;15m;30m;1h;4h;1d;1w |
| startTime | integer | 否 | N/A | Start time |  |
| endTime | integer | 否 | N/A | End time |  |
| limit | integer | 否 | N/A | Limit |  |
