**类型:** GET

**描述:** `/future/market/v1/public/symbol/detail`

#### 注意：

此方法不需要签名.

### 参数

| 名称   | 类型   | 必需 | 默认值值值值 | 描述  | 范围 |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | 字符串 | 是      | N/A     | Trading pair |        |
