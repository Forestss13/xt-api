**类型:** GET

**描述:** `/future/market/v1/public/contract/risk-balance`

#### 注意：

此方法不需要签名。

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true |  | 交易对 |  |
| direction | string | false | NEXT | 方向（PREV：上一页；NEXT：下一页） | PREV;NEXT |
| id | integer | false | N/A | id |  |
| limit | integer | false | 10 | 限制数量 |  |
