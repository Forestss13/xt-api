**类型:** GET

**描述:** `/future/market/v1/public/q/行情/book`

#### 注意：

此方法不需要签名.

### 参数

| 名称   | 类型   | 必需 | 默认值值值值 | 描述  | 范围 |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | 字符串 | 是      | N/A     | Trading pair |        |

### 响应 示例

import CodeBlock from '@theme/CodeBlock';

<CodeBlock language="json" title="响应">
  {`{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "ap": "", // ask price
    "aq": "", // ask amount
    "bp": "", // bid price
    "bq": "", // bid amount
    "s": "",  // Trading pair
    "t": 0    // Time
  },
  "returnCode": 0
}`}
</CodeBlock>
