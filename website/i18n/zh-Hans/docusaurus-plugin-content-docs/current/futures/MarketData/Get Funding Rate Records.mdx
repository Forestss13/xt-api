**类型:** GET

**描述:** `/future/market/v1/public/q/funding-rate-record`

#### 注意：

此方法不需要签名.

### 参数

| 名称 | 类型 | 必需 | 默认值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 否 | N/A | Trading pair |  |
| direction | 字符串 | 否 | NEXT | Direction (PREV: Previous page; NEXT: Next page) | PREV;NEXT |
| id | integer | 否 | N/A | id |  |
| limit | integer | 否 | 10 | Limit |  |

### 响应 示例
