**类型:** POST

**描述:** `/future/trade/v1/order/update`

> Content-类型 = `application/x-www-form-urlencoded && application/json`

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| orderId | 数字 | 是 |  | 订单 ID |  |
| price | 数字 | 是 |  | Target price |  |
| origQty | 数字 | 是 |  | Target quantity (cont) |  |
| triggerProfitPrice | 数字 | 否 | N/A | Profit target price |  |
| triggerStopPrice | 数字 | 否 | N/A | Stop-Loss price |  |
| triggerPrice类型 | 字符串 | 否 | LATEST_PRICE | Trigger price 类型 | INDEX_PRICE; MARK_PRICE; LATEST_PRICE |
| profitDelegate订单类型 | 字符串 | 否 | N/A | Take-Profit order 类型 | LIMIT; MARKET |
| profitDelegateTimeInForce | 字符串 | 否 | N/A | Take-Profit order validity 方法 | GTC; IOC; FOK; GTX |
| profitDelegatePrice | 数字 | 否 | N/A | Take-Profit order price |  |
| stopDelegate订单类型 | 字符串 | 否 | N/A | Stop-Loss order 类型 | LIMIT; MARKET |
| stopDelegateTimeInForce | 字符串 | 否 | N/A | Stop-Loss order validity 方法 | GTC; IOC; FOK; GTX |
| stopDelegatePrice | 数字 | 否 | N/A | Stop-Loss order price |  |
| followUp订单 | 布尔值 | 否 | N/A | If 是, indicates chase order |  |

### Limit Flow Rules

200/s/apikey
