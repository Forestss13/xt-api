**类型:** POST

**描述:** `/future/trade/v1/order/list`

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| client订单Id | 字符串 | 否 | N/A | Client order ID |  |
| page | integer | 否 | 1 | Page |  |
| size | integer | 否 | 10 | Quantity of a single page |  |
| startTime | integer | 否 | N/A | Start time |  |
| endTime | integer | 否 | N/A | End time |  |
| state | 字符串 | 否 | NEW | 订单 state: `NEW` (New order, unfilled); `PARTIALLY_FILLED` (Partial deal); `PARTIALLY_CANCELED` (Partial revocation); `FILLED` (Filled); `CANCELED` (Cancelled); `REJECTED` (订单 failed); `EXPIRED` (Expired); `UNFINISHED` (Unfinished); `HISTORY` (History) | PREV; NEXT |
| symbol | 字符串 | 否 | N/A | Trading pair |  |

### Limit Flow Rules

200/s/apikey
