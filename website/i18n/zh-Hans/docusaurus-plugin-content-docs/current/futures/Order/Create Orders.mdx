**类型:** POST

**描述:** `/future/trade/v1/order/create`

> Content-类型 = `application/x-www-form-urlencoded` && `application/json` 注意：此方法不需要签名.

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| client订单Id | 字符串 | 否 | N/A | Client order ID |  |
| symbol | 字符串 | 是 | - | Trading pair |  |
| orderSide | 字符串 | 是 | N/A | 订单 side | BUY;SELL |
| order类型 | 字符串 | 是 | N/A | 订单 类型 | LIMIT;MARKET |
| origQty | 数字 | 是 | N/A | Quantity (Cont) |  |
| price | 数字 | 否 | N/A | Price |  |
| timeInForce | 字符串 | 否 | GTC | Valid way | GTC;IOC;FOK;GTX |
| triggerProfitPrice | 数字 | 否 | N/A | Stop profit price |  |
| triggerStopPrice | 数字 | 否 | N/A | Stop loss price |  |
| positionSide | 字符串 | 是 | N/A | Position side | LONG;SHORT |

### OrigQty Calculation Formula

**Formula**

**Explain**

- `Truncate`: take the integer part
- `Balance`: (walletBalance - open订单MarginFrozen), api: `/future/user/v1/compat/balance/list`
- `Percent`: user input, e.g. `0.2`
- `Leverage`: leverage you want, e.g. `20`
- `Mark_price`: current symbol mark price, e.g. `88888 (btc_usdt)`
- `Contract_size`: contractSize, api: `/future/market/v1/public/symbol/detail`, contract multiplier (face value)

**示例** truncate(10000 _ 0.2 _ 20 / 88888 / 0.0001) = 4500
