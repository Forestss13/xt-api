**类型:** POST

**描述:** `/future/trade/v1/order/detail`

### 参数

| 名称    | 类型    | 必需 | 默认值值值值值值值值 | 描述 | 范围 |
| ------- | ------- | --------- | ------- | ----------- | ------ |
| orderId | integer | 是      | N/A     | 订单 ID    |        |

### Limit Flow Rules

200/s/apikey
