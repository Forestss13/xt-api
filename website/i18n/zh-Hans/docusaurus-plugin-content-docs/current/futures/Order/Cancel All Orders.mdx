**类型:** POST

**描述:** `/future/trade/v1/order/cancel-all`

### 限流规则

200/s/apikey

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true | N/A | 交易对（传入 `""` 则撤销所有交易对的订单） |  |
