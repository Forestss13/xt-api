**类型:** GET

**描述:** `/future/market/v1/public/cg/orderbook`

> Content-Type = application/x-www-form-urlencoded

### 注意事项：

此接口无需签名认证。

### 请求参数

| 参数名 | 类型   | 是否必填 | 默认值 | 描述                    | 取值范围 |
| ------ | ------ | -------- | ------ | ----------------------- | -------- |
| symbol | string | 是       | 无     | 交易对，如 BTC-USDT     |          |
| level  | int    | 否       | 50     | 深度档位                | 1-200    |
