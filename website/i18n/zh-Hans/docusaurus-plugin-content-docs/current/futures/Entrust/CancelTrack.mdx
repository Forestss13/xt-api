**类型:** POST

**描述:** `/future/trade/v1/entrust/cancel-track`

**备注:** Content-Type = application/x-www-form-urlencoded

### 限流规则

200/s/apikey

### 参数

| 名称    | 类型    | 必填 | 默认值 | 描述     | 范围 |
| ------- | ------- | ---- | ------ | -------- | ---- |
| trackId | integer | true | N/A    | 跟踪单ID |      |

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
```
