**类型:** POST

**描述:** `/future/trade/v1/entrust/cancel-profit-stop`

**备注:** Content-Type = application/x-www-form-urlencoded && application/json

### 限流规则

200/s/apikey

### 参数

| 名称     | 类型    | 必填 | 默认值 | 描述           | 范围 |
| -------- | ------- | ---- | ------ | -------------- | ---- |
| profitId | integer | true | N/A    | 止盈止损订单ID |      |

{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
