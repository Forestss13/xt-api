**类型:** POST

**描述:** `/future/trade/v1/entrust/track-list`

### 限流规则

200/s/apikey

### 参数

| 名称      | 类型    | 必填  | 默认值 | 描述     |
| --------- | ------- | ----- | ------ | -------- |
| page      | integer | false | 1      | 页码     |
| size      | integer | false | 10     | 每页数量 |
| endTime   | integer | false | N/A    | 结束时间 |
| startTime | integer | false | N/A    | 开始时间 |
| symbol    | string  | false | N/A    | 交易对   |

### 示例代码

```java
public void getTrackDetail() {
    String text = HttpUtil.get(URL + "/data/api/future/trade/v1/entrust/track-list");
    System.out.println(text);
}
```
