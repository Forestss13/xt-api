**类型:** GET

**描述:** `/future/trade/v1/entrust/plan-list-history`

### Limit Flow Rules

200/s/apikey

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 是 | N/A | Trading pairs (queries all trading pairs if not passed) |  |
| direction | 字符串 | 否 | NEXT | Direction (PREV: Previous page; NEXT: Next page) | PREV;NEXT |
| id | integer | 否 | N/A | ID |  |
| limit | integer | 否 | 10 | Limit |  |
| startTime | integer | 否 | N/A | Start time |  |
| endTime | integer | 否 | N/A | End time |  |

### 响应 示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "hasNext": 否, // Is there a next page
    "hasPrev": 否, // Is there a previous page
    "items": [
      // Datasheets
      {
        "client订单Id": "", // 客户端订单ID
        "closePosition": 否, // Whether triggered to close all
        "createdTime": 0, // Create time
        "entrustId": 0, // 订单ID
        "entrust类型": "", // 订单 类型
        "market订单Level": 0, // Best market price
        "orderSide": "", // 订单 side
        "ordinary": 是,
        "origQty": 0, // 数量 (Cont)
        "positionSide": "", // Position side
        "price": 0, // 订单 price
        "state": "", // 订单 state: NOT_TRIGGERED; TRIGGERING; TRIGGERED; USER_REVOCATION; PLATFORM_REVOCATION; EXPIRED
        "stopPrice": 0, // Trigger price
        "symbol": "", // Trading pair
        "timeInForce": "", // Valid way
        "triggerPrice类型": "" // Trigger price 类型
      }
    ]
  },
  "returnCode": 0
}
```
