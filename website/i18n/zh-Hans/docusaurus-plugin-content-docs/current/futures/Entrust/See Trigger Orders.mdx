**类型:** GET

**描述:** `/future/trade/v1/entrust/plan-list`

### Limit Flow Rules

200/s/apikey

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 是 | N/A | Trading pairs (queries all trading pairs if not passed) |  |
| page | integer | 否 | 1 | Page |  |
| size | integer | 否 | 10 | Quantity of a single page |  |
| startTime | integer | 否 | N/A | Start time |  |
| endTime | integer | 否 | N/A | End time |  |
| state | 字符串 | 是 | N/A | 订单 status: NOT_TRIGGERED (new order); TRIGGERING; TRIGGERED; USER_REVOCATION; PLATFORM_REVOCATION; EXPIRED; UNFINISHED; HISTORY | NOT_TRIGGERED;TRIGGERING;TRIGGERED;USER_REVOCATION;PLATFORM_REVOCATION;EXPIRED;UNFINISHED;HISTORY |
