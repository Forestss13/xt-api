**类型:** POST

**描述:** `/future/trade/v1/entrust/create-profit`

**Remark:** Content-类型 = application/x-www-form-urlencoded && application/json

### Limit Flow Rules

200/s/apikey

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | 字符串 | 否 | N/A | Trading pair |  |
| origQty | integer | 是 |  | Quantity (Cont) |  |
| triggerProfitPrice | integer | 是 |  | TP trigger price |  |
| triggerStopPrice | integer | 是 |  | SL trigger price |  |
| expireTime | integer | 是 |  | Expiration time |  |
| positionSide | 字符串 | 是 |  | Position side | LONG;SHORT |
| profitDelegate订单类型 | 字符串 | 是 |  | Take profit order 类型 | LIMIT, MARKET |
| profitDelegateTimeInForce | 字符串 | 是 |  | Take profit order time in force | GTC, FOK, IOC, GTX |
| profitDelegatePrice | 数字 | 否 |  | Take profit order price |  |
| stopDelegate订单类型 | 字符串 | 是 |  | Stop loss order 类型 | LIMIT, MARKET |
| stopDelegateTimeInForce | 字符串 | 是 |  | Stop loss order time in force | GTC, FOK, IOC, GTX |
| stopDelegatePrice | 数字 | 否 |  | Stop loss order price |  |

### 请求 示例

```java title="Java"
public void getKLine() {
```
