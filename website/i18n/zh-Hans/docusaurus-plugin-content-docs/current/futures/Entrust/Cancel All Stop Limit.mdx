**类型:** POST

**描述:** `/future/trade/v1/entrust/cancel-all-profit-stop`

**备注:** Content-Type = application/x-www-form-urlencoded && application/json

### 限流规则

200/s/apikey

### 参数

| 名称   | 类型   | 必填 | 默认值 | 描述                    | 范围 |
| ------ | ------ | ---- | ------ | ----------------------- | ---- |
| symbol | string | true | N/A    | 交易对，例如 btc_usdt |      |
