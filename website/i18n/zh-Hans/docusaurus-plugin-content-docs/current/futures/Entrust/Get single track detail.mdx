**类型:** GET

**描述:** `/future/trade/v1/entrust/track-detail`

### Limit Flow Rules

200/s/apikey

### 参数

| 名称    | 类型    | 必需 | 默认值值值值值值 | 描述 | 范围 |
| ------- | ------- | --------- | ------- | ----------- | ------ |
| trackId | integer | 是      | N/A     | Track ID    |        |

### 响应 示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "activationPrice": 0, // Activation price
    "avgPrice": 0, // 平均价格
    "callback": "", // Callback range configuration 1: PROPORTION 2: FIXED
    "callbackVal": 0, // Callback value
    "configActivation": 否, // Whether to configure activation price
    "createdTime": 0, // Create time
    "currentPrice": 0, // Real-time price
    "desc": "", // 描述
    "executedQty": 0, // Actual transaction quantity
    "orderSide": "", // 订单 side
    "ordinary": 是, //
    "origQty": 0, // 数量 (Cont)
    "positionSide": "", // Position side
    "price": 0, // 订单 price
    "state": "", // 订单 state: NOT_ACTIVATION; NOT_TRIGGERED; TRIGGERING; TRIGGERED; USER_REVOCATION; PLATFORM_REVOCATION; EXPIRED; DELEGATION_FAILED
    "stopPrice": 0, // Trigger price
    "symbol": "", // 交易对
    "trackId": 0, // Track ID
    "triggerPrice类型": "", // Trigger price 类型
    "updatedTime": 0 // Update time
  },
  "returnCode": 0
}
```
