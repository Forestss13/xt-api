**Remark:** Content-类型 = application/x-www-form-urlencoded && application/json

### Limit Flow Rules

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值 | 描述 | 范围 |

| triggerStopPrice | 数字 | 否 | N/A | SL trigger price |  |
