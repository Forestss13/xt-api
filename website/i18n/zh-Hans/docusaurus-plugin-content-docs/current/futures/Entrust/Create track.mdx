**类型:** POST

**描述:** `/future/trade/v1/entrust/create-track`

**Remark:** Content-类型 = application/x-www-form-urlencoded

### Limit Flow Rules

200/s/apikey

### 参数

| 名称 | 类型 | 必需 | 默认值值值值值值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| callback | 字符串 | 是 | N/A | Callback range configuration | FIXED;PROPORTION |
| callbackVal | 数字 | 是 | N/A | Callback value | Greater than 0 |
| orderSide | 字符串 | 是 | N/A | 订单 side | BUY;SELL |
| origQty | 数字 | 是 | N/A | Original quantity (count) |  |
| positionSide | 字符串 | 是 | N/A | Position side | BOTH;LONG;SHORT |
| position类型 | 字符串 | 是 | N/A | Position 类型 | CROSSED;ISOLATED |
| symbol | 字符串 | 是 | N/A | Trading pair |  |
| triggerPrice类型 | 字符串 | 是 | N/A | Trigger price 类型 | INDEX_PRICE;MARK_PRICE;LATEST_PRICE |
| activationPrice | 数字 | 否 | N/A | Activation price |  |
| clientMedia | 字符串 | 否 | N/A | Client media |  |
| clientMediaChannel | 字符串 | 否 | N/A | Client media channel |  |
| client订单Id | 字符串 | 否 | N/A | Client order id |  |
| expireTime | integer | 否 | N/A | Expiration time |  |
