**类型:** GET

**描述:** `/future/trade/v1/entrust/plan-detail`

### Limit Flow Rules

200/s/apikey

### 参数

| 名称      | 类型    | 必需 | 默认值值值值值值 | 描述 | 范围 |
| --------- | ------- | --------- | ------- | ----------- | ------ |
| entrustId | integer | 是      | N/A     | 订单 ID    |        |

### 响应 示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "closePosition": 否, // Whether triggered to close all
    "createdTime": 0, // Create time
    "entrustId": 0, // 订单ID
    "entrust类型": "", // 订单 类型
    "market订单Level": 0, // Best market price
    "orderSide": "", // 订单 side
    "ordinary": 是,
    "origQty": 0, // 数量 (Cont)
    "positionSide": "", // Position side
    "price": 0, // 订单 price
    "state": "", // 订单 state: NOT_TRIGGERED; TRIGGERING; TRIGGERED; USER_REVOCATION; PLATFORM_REVOCATION; EXPIRED; UNFINISHED; HISTORY
    "stopPrice": 0, // Trigger price
    "symbol": "", // Trading pair
    "timeInForce": "", // Time in force
    "triggerPrice类型": "" // Trigger price 类型
  },
  "returnCode": 0
}
```
