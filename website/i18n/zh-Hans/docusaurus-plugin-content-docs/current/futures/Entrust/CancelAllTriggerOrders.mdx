**类型:** POST

**描述:** `/future/trade/v1/entrust/cancel-all-plan`

**Remark:** Content-类型 = application/x-www-form-urlencoded && application/json

### Limit Flow Rules

200/s/apikey

### 参数

| 名称   | 类型   | 必需 | 默认值值值值值值 | 描述                 | 范围 |
| ------ | ------ | --------- | ------- | --------------------------- | ------ |
| symbol | 字符串 | 是      |         | Trading pair. e.g. btc_usdt |        |

### 请求 示例

```java title="Java"
public void getKLine() {
    String text = HttpUtil.get(URL + "/data/api/future/trade/v1/getKLine?market=btc_usdt&类型=1min&since=0");
    System.out.println(text);
}
```
