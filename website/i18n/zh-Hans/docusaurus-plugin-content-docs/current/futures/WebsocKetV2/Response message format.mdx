### 响应消息格式

本节说明与 WebSocket API 交互时的响应消息格式。

### 响应格式

```javascript
{
    "id": "{id}",   // 回调 ID
    "code": 1,      // 结果 0=成功; 1=失败; 2=listenKey 无效
    "msg": ""
}
```

### 响应示例（成功）

```javascript
{
  "id": "123",
  "code": 0,
  "msg": "success"
}
```
