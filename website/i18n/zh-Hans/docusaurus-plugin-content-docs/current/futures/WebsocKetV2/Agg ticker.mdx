### 请求格式

```
agg_ticker@{symbol}
```

- **示例**：`agg_ticker@btc_usdt`
- **推送频率**：1000ms

---

### 响应示例

```json title="响应"
{
  "topic": "agg_ticker",
  "event": "agg_ticker@btc_usdt",
  "data": {
    "s": "btc_index", // 交易对
    "o": "49000", // 开盘价
    "c": "50000", // 收盘价
    "h": "0.1", // 最高价
    "l": "0.1", // 最低价
    "a": "0.1", // 成交量
    "v": "0.1", // 成交额
    "ch": "0.21", // 涨跌幅
    "i": "0.21", // 指数价格
    "m": "0.21", // 标记价格
    "bp": "0.21", // 买一价
    "ap": "0.21", // 卖一价
    "t": 123124124 // 时间戳
  }
}
```
