### 请求格式

```
depth@{symbol},{levels},{interval}
```

- **档位选项**：`5`、`10`、`20`、`50`
- **时间间隔选项**：`100`、`250`、`500`、`1000`（默认频率：1000ms）
- **示例**：`depth@btc_usdt,50,1000ms`

---

### 响应示例

```json title="响应"
{
  "topic": "depth",
  "event": "depth@btc_usdt,20",
  "data": {
    "id": "1234", // lastUpdateId
    "s": "btc_index", // 交易对
    "a": [
      ["50000", "0.1"],
      ["50001", "0.2"]
    ], // 卖单队列 [价格, 数量]
    "b": [
      ["49999", "0.1"],
      ["48888", "0.2"]
    ], // 买单队列
    "t": 123456789 // 时间戳
  }
}
```
