### 请求 格式

```
fund_rate@{symbol}
```

- **example**: `fund_rate@btc_usdt`
- **rate**: 60s

---

### 响应 示例

```json title="响应"
{
  "topic": "fund_rate",
  "event": "fund_rate@btc_usdt",
  "data": {
    "s": "btc_usdt", // Trading pair
    "r": "0.01", // Fund fee
    "t": 123124124 // timestamp
  }
}
```
