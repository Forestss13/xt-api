### 请求格式

```
fund_rate@{symbol}
```

- **示例**：`fund_rate@btc_usdt`
- **推送频率**：60s

---

### 响应示例

```json title="响应"
{
  "topic": "fund_rate",
  "event": "fund_rate@btc_usdt",
  "data": {
    "s": "btc_usdt", // 交易对
    "r": "0.01", // 资金费率
    "t": 123124124 // 时间戳
  }
}
```
