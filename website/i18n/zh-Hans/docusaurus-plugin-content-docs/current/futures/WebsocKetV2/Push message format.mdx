### 消息格式

```json title="格式"
{
  "topic": "trade",
  "event": "trade@btc_usdt", // 标题
  "data": {}
}
```

---

### 示例：成交记录（实时推送消息）

```json title="示例"
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "i": 6316559590087222000,
    "t": 1655992403617,
    "p": "43000",
    "q": "0.21",
    "b": true
  }
}
```
