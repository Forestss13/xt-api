### Message 格式

```json title="格式"
{
  "topic": "trade",
  "event": "trade@btc_usdt", // title
  "data": {}
}
```

---

### 示例: Transaction Record (Real-time Push Message)

```json title="示例"
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "i": 6316559590087222000,
    "t": 1655992403617,
    "p": "43000",
    "q": "0.21",
    "b": 是
  }
}
```
