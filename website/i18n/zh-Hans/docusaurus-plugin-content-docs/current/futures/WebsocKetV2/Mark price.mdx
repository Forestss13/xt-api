### 请求 格式

```
mark_price@{symbol}
```

- **example**: `mark_price@btc_usdt`
- **rate**: 1000ms

---

### 响应 示例

```json title="响应"
{
  "topic": "mark_price",
  "event": "mark_price@btc_usdt",
  "data": {
    "s": "btc_usdt", // trading pair
    "p": "50000", // price
    "t": 123124124 // timestamp
  }
}
```
