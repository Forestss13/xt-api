**类型** GET

**描述:** /future/user/v1/compat/balance/list

#### **Limit Flow Rules**

200/s/apikey

### 参数

| 名称           | 类型   | 必需 | 默认值值 | 描述 |
| -------------- | ------ | --------- | ------- | ----------- |
| 查询参数AccountId | 字符串 | 否     | N/A     | Account ID  |
