**类型** GET

**接口说明：** /future/user/v1/compat/balance/list

#### **限流规则**

200次/秒/apikey

### 参数

| 名称           | 类型   | 必填  | 默认值 | 说明    |
| -------------- | ------ | ----- | ------ | ------- |
| queryAccountId | string | 否    | N/A    | 账户ID  |
