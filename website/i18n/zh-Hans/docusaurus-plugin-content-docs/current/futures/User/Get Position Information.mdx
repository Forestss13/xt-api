**类型** GET

**描述:** `/future/user/v1/position/list`

**参数**

| 名称 | 类型 | 必需 | 默认值值 | 描述 |
| --- | --- | --- | --- | --- |
| symbol | 字符串 | No | N/A | Trading pair (queries position information for all trading pairs when not provided) |

### Limit Flow Rules

200/s/apikey

### 响应 示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "autoMargin": 否, // Whether to automatically call margin
      "availableCloseSize": 0, // Available quantity (Cont)
      "close订单Size": 0, // Pending order quantity (Cont)
      "entryPrice": 0, // Open position average price
      "isolatedMargin": 0, // Isolated Margin
      "leverage": 0, // Leverage
      "open订单MarginFrozen": 0, // Occupied open position margin
      "positionSide": "", // Position side
      "positionSize": 0, // Position quantity (Cont)
      "position类型": "", // Position 类型
      "实时izedProfit": 0, // Realized profit and loss
      "symbol": "" // Trading pair
    }
  ],
  "returnCode": 0
}
```
