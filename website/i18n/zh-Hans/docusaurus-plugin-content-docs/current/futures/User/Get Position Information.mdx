**类型** GET

**接口说明：** `/future/user/v1/position/list`

**参数**

| 名称 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| symbol | string | 否 | N/A | 交易对（不传时查询所有交易对的持仓信息） |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "autoMargin": false, // 是否自动追加保证金
      "availableCloseSize": 0, // 可平数量（张）
      "closeOrderSize": 0, // 挂单数量（张）
      "entryPrice": 0, // 平均开仓价格
      "isolatedMargin": 0, // 逐仓保证金
      "leverage": 0, // 杠杆
      "openOrderMarginFrozen": 0, // 占用开仓保证金
      "positionSide": "", // 持仓方向
      "positionSize": 0, // 持仓数量（张）
      "positionType": "", // 持仓类型
      "realizedProfit": 0, // 已实现盈亏
      "symbol": "" // 交易对
    }
  ],
  "returnCode": 0
}
```
