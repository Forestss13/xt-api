**类型** GET

**接口说明：** `/future/user/v1/position`

**参数**

| 名称 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| symbol | string | 否 | N/A | 交易对（不传时查询所有交易对的持仓信息） |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "args": [],
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "autoMargin": false, // 是否自动追加保证金
      "availableCloseSize": 0, // 可平数量（张）
      "breakPrice": 0, // 爆仓价格
      "calMarkPrice": 0, // 计算的标记价格
      "closeOrderSize": 0, // 开仓委托数量（张）
      "contractType": "", // 合约类型：PERPETUAL（永续合约）、PREDICT（预测合约）
      "entryPrice": 0, // 平均开仓价格
      "floatingPL": 0, // 未实现盈亏
      "isolatedMargin": 0, // 逐仓保证金
      "leverage": 0, // 杠杆倍数
      "openOrderMarginFrozen": 0, // 开仓委托占用保证金
      "openOrderSize": 0, // 开仓委托占用
      "positionSide": "", // 持仓方向
      "positionSize": 0, // 持仓数量（张）
      "positionType": "", // 持仓类型：CROSSED（全仓）；ISOLATED（逐仓）
      "profitId": 0, // 止盈止损ID
      "realizedProfit": 0, // 已实现盈亏
      "symbol": "", // 交易对
      "triggerPriceType": "", // 触发价格类型 1：指数价格 2：标记价格（合理价格）；3：最新价格
      "triggerProfitPrice": 0, // 止盈触发价格
      "triggerStopPrice": 0, // 止损触发价格
      "welfareAccount": true
    }
  ],
  "returnCode": 0
}
```
