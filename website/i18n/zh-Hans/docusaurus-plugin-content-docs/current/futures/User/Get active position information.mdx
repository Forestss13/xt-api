**类型** GET

**描述:** `/future/user/v1/position`

**参数**

| 名称 | 类型 | 必需 | 默认值值 | 描述 |
| --- | --- | --- | --- | --- |
| symbol | 字符串 | No | N/A | Trading pairs (queries position information for all trading pairs when not provided) |

### Limit Flow Rules

200/s/apikey

### 响应 示例

```json title="响应"
{
  "error": {
    "args": [],
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "autoMargin": 否, // Whether to automatically call 保证金账户
      "availableCloseSize": 0, // Available quantity (Cont)
      "breakPrice": 0, // Blowout price
      "calMarkPrice": 0, // Calculated mark price
      "close订单Size": 0, // 数量 of open order (Cont)
      "contract类型": "", // Contract 类型s: PERPETUAL (Perpetual Contract), PREDICT (Predict Contract)
      "entryPrice": 0, // Average opening price
      "floatingPL": 0, // Un实时ized profit or loss
      "isolatedMargin": 0, // Warehouse-by-warehouse 保证金账户
      "杠杆账户": 0, // Leverage ratio
      "open订单MarginFrozen": 0, // Occupation of deposit for opening order
      "open订单Size": 0, // Opening warehouse orders occupied
      "positionSide": "", // Position direction
      "positionSize": 0, // Position quantity (Cont)
      "position类型": "", // Position 类型: CROSSED (full position); ISOLATED (warehouse by warehouse)
      "profitId": 0, // Take profit and stop loss id
      "实时izedProfit": 0, // Realized profit and loss
      "symbol": "", // Trading pair
      "triggerPrice类型": "", // Trigger price 类型 1. 指数价格 2: 标记价格 (reasonable price); 3: Latest price
      "triggerProfitPrice": 0, // Take profit trigger price
      "triggerStopPrice": 0, // Stop loss trigger price
      "welfareAccount": 是
    }
  ],
  "returnCode": 0
}
```
