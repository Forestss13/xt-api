**类型** GET

**接口说明：** `/v4/public/lever/symbol`

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": [
    {
      "symbolId": 123, // 现货交易对ID
      "symbol": "btc_usdt", // 现货交易对
      "buyCurrencyId": 123, // 买入币种ID
      "buyCurrency": "btc", // 买入币种
      "sellCurrencyId": 234, // 卖出币种ID
      "sellCurrency": "usdt", // 卖出币种
      "maxLoanAmountBuy": "", // 买入币种最大可借额度
      "maxLoanAmountSell": "", // 卖出币种最大可借额度
      "maxLeverage": "", // 最大借贷倍数
      "dailyInterestRate": "", // 日利率
      "liquidationRate": "", // 爆仓风险率
      "tipsRate": "" // 提醒系数
    }
  ]
}
```
