**类型** GET

**接口说明：** `/future/user/v1/balance/detail`

**参数**

| 名称 | 类型   | 必填 | 默认值 | 说明 |
| ---- | ------ | ---- | ------ | ---- |
| coin | string | 是   | N/A    | 币种 |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "availableBalance": 0, //可用余额
    "coin": "", //币种
    "isolatedMargin": 0, //冻结逐仓保证金
    "openOrderMarginFrozen": 0, //冻结订单
    "crossedMargin": 0, //全仓保证金
    "bonus": 0, //赠金
    "coupon": 0, //优惠券
    "walletBalance": 0 //余额
  },
  "returnCode": 0
}
```
