**类型** GET

**描述:** `/future/user/v1/balance/detail`

**参数**

| 名称 | 类型   | 必需 | 默认值值 | 描述 |
| ---- | ------ | --------- | ------- | ----------- |
| coin | 字符串 | Yes       | N/A     | 货币    |

### Limit Flow Rules

200/s/apikey

### 响应 示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "availableBalance": 0, //Available balance
    "coin": "", //货币
    "isolatedMargin": 0, //Frozen isolated margin
    "open订单MarginFrozen": 0, //Frozen order
    "crossedMargin": 0, //Crossed Margin
    "bonus": 0, //Bonus
    "coupon": 0, //Coupon
    "walletBalance": 0 //Balance
  },
  "returnCode": 0
}
```
