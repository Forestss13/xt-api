### 说明

**GET** `/future/trade/v1/entrust/track-list-history`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| direction | string | 否 | NEXT | 方向（PREV：上一页；NEXT：下一页） | PREV；NEXT |
| limit | integer | 否 | 10 | 限制数量 |  |
| id | integer | 否 | N/A |  |  |
| endTime | integer | 否 | N/A | 结束时间 |  |
| startTime | integer | 否 | N/A | 开始时间 |  |
| symbol | string | 否 | N/A | 交易对 |  |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "hasNext": true, // 是否有下一页

    "items": [
      {
        "activationPrice": 0, // 激活价格
        "avgPrice": 0, // 平均价格
        "callback": "", // 回调幅度配置 1：PROPORTION（比例） 2：FIXED（固定）
        "callbackVal": 0, // 回调值
        "configActivation": false, // 是否配置激活价格
        "createdTime": 0, // 创建时间
```
