**类型** POST

**描述:** `/future/user/v1/position/margin`

**参数**

| 名称 | 类型 | 必需 | 默认值值 | 描述 |
| --- | --- | --- | --- | --- |
| symbol | 字符串 | Yes | N/A | Trading pair |
| margin | 数字 | No | N/A | Quantity |
| positionSide | 字符串 | No | N/A | Position side (LONG, SHORT) |
| 类型 | 字符串 | No | N/A | Adjust direction (ADD: add isolated margin, SUB: reduce isolated margin) |

### Limit Flow Rules

200/s/apikey

    "code": "",

  },
