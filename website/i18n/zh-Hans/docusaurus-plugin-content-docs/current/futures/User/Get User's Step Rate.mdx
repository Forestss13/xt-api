**类型** GET

**接口说明：** `/future/user/v1/user/step-rate`

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "success",
  "result": {
    "specialType": false, // 是否为特殊账户
    "vipProType": true, // 是否为专业费率
    "stepRateProName": "VIP_PRO_1", // 专业费率名称
    "discountLevel": 2, // 折扣级别
    "makerFee": "0.0015", // Maker手续费率
    "takerFee": "0.0025", // Taker手续费率
    "levelReturnDay": 30, // 当前级别保留天数
    "totalTradeVolume": "1250000.50", // 过去29天+今天的交易量（每10分钟更新）
    "uBasedTotalTradeVolume": "850000.75", // 过去29天+今天的USDT保证金交易量
    "coinBasedTotalTradeVolume": "400000.25", // 过去29天+今天的币本位交易量
    "walletBalance": "50000.00", // 账户权益（USDT，每10分钟更新）：钱包余额+未实现盈亏
    "notProfit": "1250.50", // 持仓未实现盈亏（USDT，每10分钟更新）
    "nextLvTradeVolume": "1500000.00", // 下一级别交易量
    "lackTradeVolume": "250000.50", // 升级所需交易量

}
```
