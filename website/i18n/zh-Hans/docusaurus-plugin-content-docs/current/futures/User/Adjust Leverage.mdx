**类型** POST

**接口说明：** `/future/user/v1/position/adjust-leverage`

**参数**

| 名称         | 类型    | 必填 | 默认值 | 说明                        |
| ------------ | ------- | ---- | ------ | --------------------------- |
| symbol       | string  | 是   | N/A    | 交易对                      |
| positionSide | string  | 是   | N/A    | 持仓方向（LONG，SHORT）     |
| leverage     | integer | 是   | N/A    | 杠杆倍数                    |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
```
