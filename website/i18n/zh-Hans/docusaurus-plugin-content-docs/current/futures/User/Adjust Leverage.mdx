**类型** POST

**描述:** `/future/user/v1/position/adjust-leverage`

**参数**

| 名称         | 类型    | 必需 | 默认值值 | 描述                 |
| ------------ | ------- | --------- | ------- | --------------------------- |
| symbol       | 字符串  | Yes       | N/A     | Trading pair                |
| positionSide | 字符串  | Yes       | N/A     | Position side (LONG, SHORT) |
| leverage     | integer | Yes       | N/A     | Leverage                    |

### Limit Flow Rules

200/s/apikey

    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
