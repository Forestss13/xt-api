**类型** GET

**接口说明：** `/future/user/v1/balance/bills`

**参数**

| 名称 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| symbol | string | 是 | N/A | 交易对（不传查询所有） |
| direction | string | 否 | NEXT | 方向（PREV：上一页；NEXT：下一页） |
| id | integer | 否 | N/A | 记录ID |
| limit | integer | 否 | 10 | 限制数量 |
| startTime | integer | 否 | N/A | 开始时间 |
| endTime | integer | 否 | N/A | 结束时间 |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "hasNext": false, // 是否有下一页
    "hasPrev": false, // 是否有上一页
    "items": [
      // 数据列表
      {
        "afterAmount": 0, // 变动后余额
        "amount": 0, // 数量
        "coin": "", // 币种
        "createdTime": 0, // 时间
        "id": 0, // ID
        "side": "", // ADD：转入；SUB：转出
        "symbol": "", // 交易对
        "type": "" // EXCHANGE：划转；CLOSE_POSITION：平仓盈亏；TAKE_OVER：仓位接管；QIANG_PING_MANAGER：强平管理费；FUND：资金费用；FEE：手续费（开仓、平仓、强平）；ADL：自动减仓；MERGE：仓位合并
      }
    ]
  },
  "returnCode": 0
}
```
