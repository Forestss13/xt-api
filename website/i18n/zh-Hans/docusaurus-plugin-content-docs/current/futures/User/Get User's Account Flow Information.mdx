**类型** GET

**描述:** `/future/user/v1/balance/bills`

**参数**

| 名称 | 类型 | 必需 | 默认值值 | 描述 |
| --- | --- | --- | --- | --- |
| symbol | 字符串 | Yes | N/A | Trading pairs (queries all if not passed) |
| direction | 字符串 | No | NEXT | Direction (PREV: Previous page; NEXT: Next page) |
| id | integer | No | N/A | Record ID |
| limit | integer | No | 10 | Limit |
| startTime | integer | No | N/A | Start time |
| endTime | integer | No | N/A | End time |

### Limit Flow Rules

200/s/apikey

### 响应 示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "hasNext": 否, // Is there a next page
    "hasPrev": 否, // Is there a previous page
    "items": [
      // Datasheets
      {
        "afterAmount": 0, // Balance after change
        "amount": 0, // 数量
        "coin": "", // 货币
        "createdTime": 0, // Time
        "id": 0, // ID
        "side": "", // ADD: transfer in; SUB: transfer out
        "symbol": "", // Trading pair
        "类型": "" // EXCHANGE: transfer; CLOSE_POSITION: Offset PnL; TAKE_OVER: position takeover; QIANG_PING_MANAGER: Liquidation management fee; FUND: Fund fee; FEE: Fee (open position, liquidation, forced liquidation); ADL: Auto-deleveraging; MERGE: Position merge
      }
    ]
  },
  "returnCode": 0
}
```
