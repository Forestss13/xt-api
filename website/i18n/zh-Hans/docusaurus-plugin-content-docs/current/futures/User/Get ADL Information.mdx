**类型** GET

**接口说明：** `/future/user/v1/position/adl`

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "longQuantile": 0, //多头持仓ADL
      "shortQuantile": 0, //空头持仓ADL
      "symbol": "" //交易对
    }
  ],
  "returnCode": 0
}
```
