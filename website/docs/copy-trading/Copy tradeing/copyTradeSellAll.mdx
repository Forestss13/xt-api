## 接口

**POST** `/v1/copy-trade-order/copy-trade/order/sell-all`

#### **Limit Flow Rules**

50/s/apikey

## 请求参数

| Name | Type | Mandatory | Default | Description |
| --- | --- | --- | --- | --- |
| `leaderOrder` | bool | Yes |  | true：sell all lead order; false：sell all copy order |
| `symbol` | string | No |  | symbol |
