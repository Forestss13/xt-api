## 接口

**GET** `/v1/copy-trade-balance/public/copy-trade/leader-income-chart`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| Name        | Type   | Mandatory | Default | Description       |
| ----------- | ------ | --------- | ------- | ----------------- |
| `accountId` | number | Yes       |         | leader account id |
| `days`      | number | No        | 7       | query days        |
