## 接口

**GET** `/v1/copy-trade-account/copy-trade/my-leaders`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| Name | Type | Mandatory | Default | Description |
| --- | --- | --- | --- | --- |
| `type` | number | Yes |  | the type of lead trade |
| `direction` | string | No | NEXT | query direction |
| `limit` | number | No | 10 | number of pages |
| `id` | object | No |  | start ID，e.g. 6216559590087220004 |

## 响应示例

```json
{
  "ma": [{}],
  "mc": "string",
  "rc": 0,
  "result": {
    "hasNext": true,
    "hasPrev": true,
    "items": [
      {
        "accountId": 0, //account id
        "avatar": "string", //avatar
        "followMargin": 0, //follow amount
        "followProfit": 0, //follow profit
        "id": 0, //the id used for pagination query
        "leaderName": "string" //nickname of lead trader
      }
    ]
  }
}
```
