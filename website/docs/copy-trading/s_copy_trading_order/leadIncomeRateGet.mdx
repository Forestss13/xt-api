## 接口

**GET** `/v1/copy-trade-balance/public/copy-trade/leader-income-rate`

#### **Limit Flow Rules**

## 请求参数

| 名称        | 类型   | 必填 | 默认值 | 说明              | 范围        |
| ----------- | ------ | ---- | ------ | ----------------- | ----------- |
| `accountId` | number | 是   |        | leader account id |             |
| `days`      | number | 否   | 7      | query days        | 7,30,90,180 |
