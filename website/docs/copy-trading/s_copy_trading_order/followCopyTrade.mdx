## 接口

**POST** `/v1/copy-trade-account/copy-trade/follow`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `leaderAccountId` | number | 是 |  | leader account id |  |
| `followType` | string | 是 |  | follow type | <a href="#followType">follow type</a>, FUND;RATE |
| `followVal` | number | 是 |  | follow fund value or rate value |  |
| `followLoss` | number | 否 |  | follow loss |  |
| `followLossOperate` | string | 是 |  | follow loss operate | <a href="#followLossOperate">follow loss operate</a>, MARKET_SELL;LEADER_CLOSE;MANUAL_PROCESS |
| `triggerStopRate` | number | 否 |  | single stop loss rate |  |
| `triggerProfitRate` | number | 否 |  | single take profit rate |  |
| `autoFollowSymbol` | bool | 是 |  | auto follow symbol or not |  |
| `symbols` | string | 是 |  | copy trade symbols. e.g. xt_usdt,btc_usdt |  |
