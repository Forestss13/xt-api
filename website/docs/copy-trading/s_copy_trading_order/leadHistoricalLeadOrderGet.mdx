## 接口

**GET** `/v1/copy-trade-order/public/copy-trade/order/leader-order-history`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `leaderAccountId` | number | 是 |  | leader account id |  |
| `symbol` | string | 否 |  | trading pair eg:btc_usdt. if null, defaults to all |  |
| `direction` | string | 否 | NEXT | query direction | PREV-previous page,NEXT-next page |
| `limit` | number | 否 | 10 | number of pages | 1-100 |
| `id` | object | 否 |  | start ID，e.g. 6216559590087220004 |  |
