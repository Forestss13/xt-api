## 接口

**GET** `/v1/copy-trade-account/copy-trade/my-leaders`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `type` | number | 是 |  | the type of lead trade | <a href="#leaderTypeCn">leader type</a>, 1：current lead trades; 2：historical lead trades |
| `direction` | string | 否 | NEXT | query direction | PREV-previous page,NEXT-next page |
| `limit` | number | 否 | 10 | number of pages | 1-100 |
| `id` | object | 否 |  | start ID，e.g. 6216559590087220004 |  |
