## 接口

**POST** `/v1/copy-trade-account/copy-trade/leader-settings`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `leaderIntro` | string | 否 |  | Leader introduction |  |
| `labelIds` | string | 否 |  | Label ids, separated by commas, e.g. 1,2,3,4 |  |
| `displayEquity` | bool | 否 |  | Display total assets or not |  |
| `displayQuantity` | bool | 否 |  | Display buy quantity or not |  |
| `symbols` | string | 是 |  | Selected copy trade symbols, e.g. btc_usdt,eth_usdt |  |
| `openLeader` | bool | 否 |  | Open leader or not |  |
