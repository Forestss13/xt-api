## 接口

**GET** `/v1/copy-trade-order/copy-trade/order/cur-follower-order`

#### **Limit Flow Rules**

2/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 否 |  | Trading pair, if not filled in, represents all |  |
| `type` | number | 否 | 2 | <a href="#leaderOrderRequestType">Leader order request type</a>, 1,detail; 2,summarize |  |
