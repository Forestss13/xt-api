## 接口

**GET** `/v1/copy-trade-order/public/copy-trade/order/leader-order-history`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| Name | Type | Mandatory | Default | Description |
| --- | --- | --- | --- | --- |
| `leaderAccountId` | number | Yes |  | leader account id |
| `symbol` | string | No |  | trading pair eg:btc_usdt. if null, defaults to all |
| `direction` | string | No | NEXT | query direction |
| `limit` | number | No | 10 | number of pages |
| `id` | object | No |  | start ID，e.g. 6216559590087220004 |
