## 接口

**GET** `/v1/copy-trade-balance/copy-trade/share-profit/his-detail`

#### **Limit Flow Rules**

2/s/ip

## 请求参数

| Name | Type | Mandatory | Default | Description |
| --- | --- | --- | --- | --- |
| `time` | number | Yes |  | profit sharing time. e.g. 1717545600000 |
| `direction` | string | No | NEXT | query direction |
| `limit` | number | No | 10 | number of pages |
| `id` | object | No |  | start ID，e.g. 6216559590087220004 |

## 响应示例

```json
{
  "ma": [{}],
  "mc": "string",
  "rc": 0,
  "result": {
    "hasNext": true,
    "hasPrev": true,
    "items": [
      {
        "amount": 0, //profit sharing amount.
        "avatar": "string", //avatar of copy trader.
        "id": 0, //the id used for pagination query.
        "name": "string" //nickname of copy trader.
      }
    ]
  }
}
```
