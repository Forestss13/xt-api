## 接口

**GET** `/v1/copy-trade-order/copy-trade/order/stop-profit-loss`

#### **Limit Flow Rules**

2/s/apikey

## 请求参数

| Name | Type | Mandatory | Default | Description |
| --- | --- | --- | --- | --- |
| `symbol` | string | Yes |  | symbol |
| `leaderOrderId` | number | Yes |  | For copy order, need leaderOrderId; for lead order, need orderId |
| `triggerProfitPrice` | number | No |  | trigger profit price |
| `triggerStopPrice` | number | No |  | trigger stop price |

## 响应示例

```json
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": true
}
```
