# 跟单交易 API

欢迎使用 XT 跟单交易 API！本 API 提供了完整的跟单交易功能，支持跟随优秀交易者的策略，实现自动化交易。

## 主要功能

- **跟单策略**: 支持多种跟单策略和模式
- **风险控制**: 智能风险管理和资金分配
- **实时同步**: 毫秒级跟单延迟
- **策略管理**: 灵活的跟单参数设置

## API 特点

- **低延迟**: 毫秒级跟单响应
- **高精度**: 精确复制交易策略
- **智能风控**: 自动风险控制
- **灵活配置**: 个性化跟单设置

## 跟单模式

- **全仓跟单**: 按比例复制所有交易
- **部分跟单**: 选择性复制部分交易
- **智能跟单**: AI 优化跟单策略
- **定时跟单**: 按时间计划跟单

## 支持的市场

- 现货市场
- 期货市场
- 保证金现货
- 期权市场
