# 安装和配置

## 环境要求

- Node.js 16.0 或更高版本
- npm 8.0 或 yarn 1.22 或更高版本
- 支持 ES6+ 的现代浏览器
- 稳定的网络连接

## 安装步骤

### 1. 安装 SDK

```bash
npm install @xt/copy-trading-api
```

或者使用 yarn：

```bash
yarn add @xt/copy-trading-api
```

### 2. 导入模块

```javascript
import {CopyTradingAPI} from '@xt/copy-trading-api';
```

### 3. 初始化配置

```javascript
const copyTradingAPI = new CopyTradingAPI({
  apiKey: 'your-api-key',
  secretKey: 'your-secret-key',
  passphrase: 'your-passphrase',
  sandbox: false,
  baseURL: 'https://api.xt.com',
});
```

## 配置参数

| 参数       | 类型    | 必填 | 说明                 |
| ---------- | ------- | ---- | -------------------- |
| apiKey     | string  | 是   | API 访问密钥         |
| secretKey  | string  | 是   | API 签名密钥         |
| passphrase | string  | 是   | API 密码短语         |
| sandbox    | boolean | 否   | 是否使用沙盒环境     |
| baseURL    | string  | 否   | API 基础地址         |
| timeout    | number  | 否   | 请求超时时间（毫秒） |

## 跟单设置

### 1. 选择跟单对象

```javascript
try {
  const leaders = await copyTradingAPI.getLeaders({
    market: 'spot', // 市场类型
    minProfit: 0.1, // 最小收益率
    maxRisk: 0.05, // 最大风险
  });
  console.log('可跟单对象:', leaders);
} catch (error) {
  console.error('获取失败:', error.message);
}
```

### 2. 设置跟单参数

```javascript
try {
  const result = await copyTradingAPI.setCopyParameters({
    leaderId: 'leader123',
    copyRatio: 0.5, // 跟单比例
    maxAmount: 1000, // 最大跟单金额
    stopLoss: 0.1, // 止损比例
    takeProfit: 0.2, // 止盈比例
  });
  console.log('跟单参数设置成功:', result);
} catch (error) {
  console.error('设置失败:', error.message);
}
```

## 错误处理

```javascript
try {
  const result = await copyTradingAPI.startCopyTrading('leader123');
  console.log('跟单开始成功:', result);
} catch (error) {
  if (error.code === 'INSUFFICIENT_BALANCE') {
    console.error('余额不足，请充值');
  } else if (error.code === 'LEADER_NOT_FOUND') {
    console.error('跟单对象不存在');
  } else if (error.code === 'COPY_ALREADY_STARTED') {
    console.error('跟单已开始，请勿重复操作');
  } else {
    console.error('请求失败:', error.message);
  }
}
```
