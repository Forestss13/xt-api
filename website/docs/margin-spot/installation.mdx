# 安装和配置

## 环境要求

- Node.js 16.0 或更高版本
- npm 8.0 或 yarn 1.22 或更高版本
- 支持 ES6+ 的现代浏览器

## 安装步骤

### 1. 安装 SDK

```bash
npm install @xt/margin-spot-api
```

或者使用 yarn：

```bash
yarn add @xt/margin-spot-api
```

### 2. 导入模块

```javascript
import {MarginSpotAPI} from '@xt/margin-spot-api';
```

### 3. 初始化配置

```javascript
const marginSpotAPI = new MarginSpotAPI({
  apiKey: 'your-api-key',
  secretKey: 'your-secret-key',
  passphrase: 'your-passphrase',
  sandbox: false,
  baseURL: 'https://api.xt.com',
});
```

## 配置参数

| 参数       | 类型    | 必填 | 说明                 |
| ---------- | ------- | ---- | -------------------- |
| apiKey     | string  | 是   | API 访问密钥         |
| secretKey  | string  | 是   | API 签名密钥         |
| passphrase | string  | 是   | API 密码短语         |
| sandbox    | boolean | 否   | 是否使用沙盒环境     |
| baseURL    | string  | 否   | API 基础地址         |
| timeout    | number  | 否   | 请求超时时间（毫秒） |

## 账户设置

### 1. 开通保证金账户

```javascript
try {
  const result = await marginSpotAPI.openMarginAccount();
  console.log('保证金账户开通成功:', result);
} catch (error) {
  console.error('开通失败:', error.message);
}
```

### 2. 设置杠杆倍数

```javascript
try {
  const result = await marginSpotAPI.setLeverage({
    symbol: 'BTC/USDT',
    leverage: 5,
  });
  console.log('杠杆设置成功:', result);
} catch (error) {
  console.error('设置失败:', error.message);
}
```

## 错误处理

```javascript
try {
  const result = await marginSpotAPI.getMarginAccount();
  console.log('保证金账户信息:', result);
} catch (error) {
  if (error.code === 'INSUFFICIENT_BALANCE') {
    console.error('余额不足，请充值');
  } else if (error.code === 'RISK_LIMIT') {
    console.error('风险超限，请调整仓位');
  } else {
    console.error('请求失败:', error.message);
  }
}
```
