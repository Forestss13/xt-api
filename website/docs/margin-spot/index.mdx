# 保证金现货 API

欢迎使用 XT 保证金现货 API！本 API 提供了完整的保证金现货交易功能，支持杠杆交易、借贷管理等。

## 主要功能

- **杠杆交易**: 支持最高 10 倍杠杆交易
- **借贷管理**: 灵活的借贷和还款操作
- **风险控制**: 实时风险监控和预警
- **资金管理**: 保证金账户资金管理

## API 特点

- **高杠杆**: 支持 1-10 倍杠杆
- **灵活借贷**: 支持多种借贷币种
- **实时风控**: 动态风险控制
- **成本透明**: 清晰的费用结构

## 支持的功能

- 保证金账户管理
- 杠杆交易下单
- 借贷和还款
- 风险监控
- 资金划转
