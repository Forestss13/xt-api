### Description

`/v4/lever/balance`

### Parameters

| Name   | Type   | Mandatory | Default | Description             | Ranges |
| ------ | ------ | --------- | ------- | ----------------------- | ------ |
| symbol | string | Yes       | –       | Symbol, e.g. `BTC_USDT` | –      |

---
