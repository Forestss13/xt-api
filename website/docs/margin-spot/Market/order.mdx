**Type:** POST

**Description:** `/v4/lever/order`

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| MarketName | String | Yes | - | Trading pair, e.g. `BTC_USDT` | - |
| CoinName | String | Yes | - | Coin name, lowercase. e.g. `btc` | - |
| Amount | Number | Yes | - | Amount, e.g. `8000` | - |
| Type | String | Yes | - | [Order Type](#marginOrderType), supports: `LOAN` / `REPAY` | - |
| Media | String | No | - | Media source, e.g. `btok` | - |
| MediaChannel | String | No | - | Media channel | - |
