**Type:** GET **Description:** `/v4/public/depth`

### Parameters

| name | type | mandatory | default | description | ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true |  | trading pair eg:btc_usdt |  |
| limit | number | false | 100 | minimum number of queries is 100 | 1~500 |

#### **Limit Flow Rules**

10/s/ip
