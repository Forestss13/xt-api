**Type:** GET  
**Description:** `/v4/public/trade/recent`

### Parameters

| name   | type   | mandatory | default | description  | ranges |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | string | true      |         | trading pair |        |
| limit  | number | false     | 200     |              | 1~1000 |

#### **Limit Flow Rules**

10/s/ip

### Response Example

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": [
    {
      "i": 0,
      "t": 0,
      "p": "string",
      "q": "string",
      "v": "string",
      "b": true
    }
  ]
}
```
