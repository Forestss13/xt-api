### Request

- **format:** `depth_update@{symbol}`
- **example:** `depth_update@btc_usdt`
- **rate:** 100ms

### Response Example

```json title="Response"
{
  "topic": "depth_update",
  "event": "depth_update@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "fi": 121,
    "i": 123,
    "a": [
      ["34000", "1.2"],
      ["34001", "2.3"]
    ],
    "b": [
      ["32000", "0.2"],
      ["31000", "0.5"]
    ]
  }
}
```
