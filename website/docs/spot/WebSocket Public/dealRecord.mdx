### Request

- **format:** `trade@{symbol}`
- **example:** `trade@btc_usdt`
- **rate:** real

### Response Example

```json title="Response"
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_usdt",
    "i": 6316559590087222000,

    "oi": 6616559590087222666,
    "p": "43000",
    "q": "0.21",
    "v": "9030",
    "b": true
  }
}
```
