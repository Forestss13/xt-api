### Request

- **format:** `depth@{symbol},{levels}`
- **levels:** 5, 10, 20, 50
- **example:** `depth@btc_usdt,20`
- **rate:** 1000ms

### Response Example

```json title="Response"
{
  "topic": "depth",
  "event": "depth@btc_usdt,20",
  "data": {
    "s": "btc_usdt",
    "i": 1657699200000,
    "a": [
      ["34000", "1.2"],
      ["34001", "2.3"]
    ],
    "b": [
      ["32000", "0.2"],
      ["31000", "0.5"]
    ]
  }
}
```
