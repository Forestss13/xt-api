### Request

- **format:** `kline@{symbol},{interval}`
- **interval:** 1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 8h, 12h, 1d, 3d, 1w, 1M
- **example:** `kline@btc_usdt,5m`
- **rate:** 1000ms

### Response Example

```json title="Response"
{
  "topic": "kline",
  "event": "kline@btc_usdt,5m",
  "data": {
    "s": "btc_usdt",
    "t": 1656043200000,
    "i": "5m",
    "o": "44000",
    "c": "50000",
    "h": "52000",
    "l": "36000",
    "q": "34.2",
    "v": "230000"
  }
}
```
