**Type:** GET **Description:** `/v4/balance`

### Parameters

| name | type | mandatory | default | description | ranges |
| --- | --- | --- | --- | --- | --- |
| currency | string | true | N/A | Currency symbol (e.g. usdt) |  |

### Notes

This endpoint retrieves the asset information of a specified currency.

### Response Example

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {
    "currency": "usdt",
    "currencyId": 0,
    "frozenAmount": 0,
    "availableAmount": 0,
    "totalAmount": 0,
    "convertBtcAmount": 0 // Converted BTC amount
  }
}
```
