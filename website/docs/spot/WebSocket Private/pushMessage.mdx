### Format

```javascript title="Format (with comments)"
{
  "topic": "trade",          // channel name
  "event": "trade@btc_usdt", // event identifier with symbol
  "data": {}                 // payload
}
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {}
}
Example: transaction record (real-time push)

{
  "topic": "trade",

  "data": {

  }
}
```
