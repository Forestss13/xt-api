# 现货交易 API

欢迎使用 XT 现货交易 API！本 API 提供了完整的现货交易功能，包括市场数据查询、下单交易、账户管理等。

## 主要功能

- **市场数据**: 实时价格、深度、K 线等数据
- **交易操作**: 买入、卖出、撤单等交易功能
- **账户管理**: 余额查询、资金划转等
- **订单管理**: 订单查询、历史记录等

## API 特点

- **高性能**: 毫秒级响应，支持高频交易
- **稳定性**: 99.9% 可用性保证
- **安全性**: 多重加密，安全可靠
- **易用性**: 简洁的 REST API 设计
