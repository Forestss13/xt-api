### Endpoint

`POST /future/copytrade/order/v1/copy-trade/close-all`

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | No | - | Trading pair | - |
| closeLongShortType | string | No | - | Direction | LONG, SHORT, BOTH |
