### Endpoint

`GET /future/copytrade/user/v1/copy-trade/my-profit-history-detail`

---

### Parameters

| Name       | Type   | Mandatory | Default | Description      | Ranges     |
| ---------- | ------ | --------- | ------- | ---------------- | ---------- |
| divideTime | number | Yes       | -       | Time             | -          |
| direction  | string | No        | -       | Paging direction | NEXT, PREV |
| limit      | number | No        | -       | Page size        | 10         |
| id         | number | No        | -       | ID               | -          |

---

### Limit Flow Rules

2/s/apikey

---

    ]
  }
}
