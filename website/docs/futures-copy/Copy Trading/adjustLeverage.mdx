### Endpoint

`POST /future/copytrade/order/v1/position/adjust-leverage`

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | Yes | - | Symbol | - |
| leverage | number | Yes | - | Leverage | - |
| positionSide | enum | No | - | Direction | LONG, SHORT, BOTH |

---

### Limit Flow Rules

2/s/apikey

### Response Example

```json
{
  "returnCode": 0,
  "msgInfo": "success",
  "error": null,
```
