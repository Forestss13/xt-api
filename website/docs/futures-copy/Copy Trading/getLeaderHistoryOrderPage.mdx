### Endpoint

`GET /future/copytrade/order/v1/copy-trade/leader-order-history`

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| type | number | Yes | - | Type, 1 (detail), 2 (total) | 1, 2 |
| symbol | string | No | - | Trading symbol | - |
| direction | string | No | - | Paging direction | NEXT, PREV |
| limit | number | No | - | Page size | 10 |
| id | number | No | - | ID | - |

---

### Limit Flow Rules

2/s/apikey

---
