### Endpoint

`GET /future/copytrade/user/v1/copy-trade/my-profit-history`

---

### Parameters

| Name      | Type   | Mandatory | Default | Description | Ranges     |
| --------- | ------ | --------- | ------- | ----------- | ---------- |
| direction | string | No        | -       | direction   | NEXT, PREV |
| limit     | number | No        | -       | page size   | 10         |
| id        | number | No        | -       | id          | -          |

---

### Limit Flow Rules

2/s/apikey
