### Endpoint

`GET /future/copytrade/public/order/v1/public/copy-trade/leader-order-page`

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| type | number | No | - | Type (1: detail, 2: total) | 1,2 |
| symbol | string | No | - | Trading pair | - |
| leaderAccountId | number | Yes | - | Leader accountId | - |
| direction | string | No | - | Page direction | NEXT, PREV |
| limit | number | No | 10 | Page size | 10 |
| id | number | No | - | ID | - |

---

### Limit Flow Rules

2/s/ip

---
