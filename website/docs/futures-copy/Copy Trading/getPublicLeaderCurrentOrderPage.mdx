### Endpoint

`GET /future/copytrade/order/v1/copy-trade/follower-order-page`

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| type | number | No | - | type, 1(detail), 2(total) | 1, 2 |
| symbol | string | No | - | Symbol | - |
| direction | string | No | - | Direction | NEXT, PREV |
| limit | number | No | - | Page size | 10 |
| id | number | No | - | ID | - |

---

### Limit Flow Rules

2/s/apikey
