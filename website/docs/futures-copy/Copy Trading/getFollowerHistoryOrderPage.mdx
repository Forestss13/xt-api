**Type:** GET

**Description:** `/future/copytrade/order/v1/copy-trade/follower-order-history`

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| type | number | No | - | Type: 1 (detail), 2 (total) | 1, 2 |
| symbol | string | No | - | Trading symbol | - |
| direction | string | No | - | Page direction | NEXT, PREV |
| limit | number | No | - | Page size | 10 |
| id | number | No | - | Record ID | - |

### Limit Flow Rules

2/s/apikey
