# Futures Copy Trading

Futures Copy Trading allows users to automatically copy the trading strategies of experienced traders.

## Overview

The Futures Copy Trading system provides a comprehensive set of APIs for:

- Following successful traders
- Managing copy trading positions
- Monitoring performance and profits
- Adjusting risk parameters

## Getting Started

To get started with Futures Copy Trading, please refer to:

- [Access Description](./AccessDescription/RestApi) - API access and authentication
- [Copy Trading APIs](./Copy%20Trading/chooseLeader) - Core trading functionality

## Key Features

- **Leader Selection**: Choose from top-performing traders
- **Risk Management**: Set leverage and position limits
- **Real-time Monitoring**: Track positions and profits
- **Flexible Controls**: Start, stop, or modify copy trading anytime
