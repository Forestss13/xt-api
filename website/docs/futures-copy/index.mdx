# 期货跟单 API

欢迎使用 XT 期货跟单 API！本 API 专门为期货市场提供跟单交易功能，支持跟随优秀期货交易者的策略，实现自动化期货交易。

## 主要功能

- **期货跟单**: 专门针对期货市场的跟单功能
- **杠杆管理**: 智能杠杆控制和风险分配
- **合约选择**: 支持多种期货合约跟单
- **风险控制**: 期货特有的风险管理系统

## API 特点

- **期货专用**: 专为期货市场优化
- **高杠杆支持**: 支持高杠杆跟单
- **合约灵活**: 多种合约类型支持
- **风险可控**: 完善的风险控制机制

## 跟单模式

- **全仓跟单**: 按比例复制所有期货交易
- **部分跟单**: 选择性复制部分期货交易
- **智能跟单**: AI 优化期货跟单策略
- **定时跟单**: 按时间计划跟单

## 支持的期货类型

- 永续合约
- 交割合约
- 期权合约
- 混合合约
