# Installation & Setup

Learn how to set up and configure Futures Copy Trading.

## Prerequisites

Before using Futures Copy Trading APIs, ensure you have:

1. **API Access**: Valid API key and secret
2. **Account Setup**: Verified trading account
3. **Permissions**: Copy trading permissions enabled

## API Configuration

### Base URL

```
https://fapi.xt.com
```

### Authentication

All API requests require proper authentication headers:

- `validate-appkey`: Your API key
- `validate-timestamp`: Current timestamp
- `validate-signature`: Request signature

## Quick Start

1. **Enable Copy Trading**: Contact support to enable copy trading features
2. **Get API Credentials**: Generate API key and secret in your account
3. **Test Connection**: Use the [API Demo](./AccessDescription/apiDemo) to test connectivity
4. **Start Trading**: Begin following leaders or managing positions
