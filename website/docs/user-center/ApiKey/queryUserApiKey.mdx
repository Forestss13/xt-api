## 接口

**GET** `/v4/user/account/api-key`

## 请求参数

| Name            | Type    | Mandatory | Default | Description             |
| --------------- | ------- | --------- | ------- | ----------------------- |
| `userAccountId` | string  | No        |         | account id              |
| `keyName`       | string  | No        |         | apiKey name             |
| `isLock`        | integer | No        |         | apiKey is locked or not |
