## 接口

**GET** `/v4/user/account/api-key`

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `userAccountId` | string | 否 |  | account id |  |
| `keyName` | string | 否 |  | apiKey name |  |
| `isLock` | integer | 否 |  | apiKey is locked or not | 0：apiKey is not locked；1：apiKey is locked |

## 示例请求

### Java

```java
public String apiKeyGet(){


}
```
