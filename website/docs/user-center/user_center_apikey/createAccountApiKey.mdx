## 接口

**POST** `/v4/user/account/api-key`

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `userAccountId` | string | 是 |  | account id |  |
| `keyName` | string | 是 |  | apiKey name |  |
| `bindIps` | string | 否 |  | bind ip list, multiple can be separated by commas |  |
| `roleScopes` | string | 是 |  | permissions code | QUERY：query is enabled; TRADE：spot and margin trade is enabled; WITHDRAW：withdraw is enabled; LEVER_LOAN：margin pledged is enabled; FUTURE：future trade is enabled; TRANSFER：transfer is enabled |
| `tags` | string | 否 |  | tags |  |
