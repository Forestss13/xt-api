## 接口

**GET** `/v4/referal/invite/multi/user/info`

## 请求参数

| Name        | Type   | Mandatory | Default | Description           |
| ----------- | ------ | --------- | ------- | --------------------- |
| `uidList`   | array  | Yes       |         | User ID list          |
| `fromId`    | number | No        |         | Starting ID           |
| `direction` | string | No        | NEXT    | query direction       |
| `limit`     | int    | No        | 10      | Limit number, max 100 |
