**Description:** `/v4/user/account/vip-info`

```json title="Response"
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "vipLevel": 0, // vip level
    "vipName": "VIP0", // vip name
    "makerFeeRate": 0.002, // Maker fee rate
    "takerFeeRate": 0.001 // Taker fee rate
  }
}
```
