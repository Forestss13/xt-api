## 接口

**GET** `/v4/referal/invite/multi/user/info`

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `uidList` | array | 是 |  | User ID list |  |
| `fromId` | number | 否 |  | Starting ID |  |
| `direction` | string | 否 | NEXT | query direction | PREV, NEXT |
| `limit` | int | 否 | 10 | Limit number, max 100 | `1<=limit<=100` |
