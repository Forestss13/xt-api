## 接口

**GET** `/v4/referal/invite/agent/rebate/data`

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `startTime` | number | 否 |  | Commission deposit start time |  |
| `endTime` | number | 否 |  | Commission deposit end time |  |
| `inviteCode` | string | 否 |  | Registration invitation code |  |
| `type` | number | 是 |  | Commission type | 1-spot 2-future |
| `fromId` | number | 否 |  | Starting ID |  |
| `direction` | string | 否 | NEXT | query direction | PREV, NEXT |
| `limit` | int | 否 | 10 | Limit number, max 100 | `1<=limit<=100` |
