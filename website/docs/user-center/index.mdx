# 用户中心 API

欢迎使用 XT 用户中心 API！本 API 提供了完整的用户账户管理功能，包括用户信息、安全设置、通知管理等。

## 主要功能

- **账户管理**: 用户信息查询和修改
- **安全设置**: 密码、2FA、API 密钥管理
- **通知管理**: 邮件、短信、推送通知
- **权限控制**: 细粒度的权限管理系统

## 快速开始

1. [安装和配置](./installation)
2. [用户认证](./authentication)
3. [账户管理](./account-management)
4. [安全设置](./security-settings)

## API 特点

- **安全可靠**: 多重安全验证机制
- **权限精细**: 细粒度的权限控制
- **功能完整**: 覆盖用户管理各个方面
- **易于集成**: 标准化的 REST API

## 主要功能模块

- 用户信息管理
- 安全认证设置
- 通知偏好配置
- API 密钥管理
- 登录历史记录
- 账户活动日志

## 安全特性

- 多重身份验证 (2FA)
- API 密钥权限控制
- 登录异常检测
- 操作日志记录
- 敏感操作确认
