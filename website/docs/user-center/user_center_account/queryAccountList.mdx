## 接口

**GET** `/v4/user/account`

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `accountId` | string | 否 |  | account id |  |
| `accountName` | string | 否 |  | account name |  |
| `level` | integer | 否 |  | account level | 1:master account; 2:sub account |

## 示例请求

### Java

```java
public String accountGet(){


}
```
