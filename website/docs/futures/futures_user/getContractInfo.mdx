## 接口

**GET** `/future/user/v1/account/info`

#### **Limit Flow Rules**

200/s/apikey

## 示例请求

### Java

```java
public void getMarketConfig() {
	String text = HttpUtil.get(URL + "/data/api/user/v1/getMarketConfig");
	System.out.println(text);
}
```

## 响应示例

### Response

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "accountId": 0, //Account ID
    "allowOpenPosition": false, //Is it possible to open position
    "allowTrade": false, //Is it possible to trade
    "allowTransfer": false, //Is it possible to transfer
    "openTime": "", //Opening time
    "state": 0, //User status
    "userId": 0 //User ID
  },
  "returnCode": 0
}
```
