## 接口

**POST** `/future/user/v1/position/change-type`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称           | 类型   | 必填 | 默认值 | 说明          | 范围             |
| -------------- | ------ | ---- | ------ | ------------- | ---------------- |
| `symbol`       | string | 是   | N/A    | Trading pair  |                  |
| `positionSide` | string | 是   | N/A    | Position side | LONG;SHORT       |
| `positionType` | string | 是   | N/A    | Position type | CROSSED;ISOLATED |
