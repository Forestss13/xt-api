## 接口

**GET** `/future/user/v1/position/list`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 否 | N/A | Trading pair (see the position information of all trading pairs if don't pass parameters) |  |

## 示例请求

### Java

```java
public void getMarketConfig() {
	String text = HttpUtil.get(URL + "/data/api/user/v1/getMarketConfig");
	System.out.println(text);
}
```
