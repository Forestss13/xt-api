## 接口

**GET** `/future/user/v1/position/break-list`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 否 | N/A | Trading pair (see the margin call of all trading pairs if don't pass parameters) |  |

## 示例请求

### Java

```java
public void getMarketConfig() {
	String text = HttpUtil.get(URL + "/future/user/v1/position/break-list");
	System.out.println(text);
}
```

## 响应示例

### Response

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "breakPrice": 0, //Margin call price. 0 means no margin call
      "calMarkPrice": 0, //Mark price
      "contractType": "", //Futures type: PERPETUAL;PREDICT
      "entryPrice": 0, //Open position average price
      "isolatedMargin": 0, //Isolated Margin
      "leverage": 0, //Leverage
      "positionSide": "", //Position side:LONG;SHORT
      "positionSize": 0, //Position quantity (Cont)
      "positionType": "", //Position type:CROSSED;ISOLATED
      "symbol": "" //Symbol
    }
  ],
  "returnCode": 0
}
```
