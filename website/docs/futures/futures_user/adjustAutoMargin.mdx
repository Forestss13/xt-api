## 接口

**POST** `/future/user/v1/position/auto-margin`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 是 | N/A | Trading pair |  |
| `positionSide` | string | 是 | N/A | Position side | LONG;SHORT |
| `autoMargin` | boolean | 是 | N/A | Whether to automatically call margin | true;false |
