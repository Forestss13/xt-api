### Description

**GET** `/future/trade/v1/entrust/track-detail`

### Limit Flow Rules

200/s/apikey

### Parameters

| Name    | Type    | Mandatory | Default | Description |
| ------- | ------- | --------- | ------- | ----------- |
| trackId | integer | true      | N/A     | Track id    |

### Example Code

```java
public void getTrackDetail() {
    String text = HttpUtil.get(URL + "/data/api/future/trade/v1/entrust/track-detail");
    System.out.println(text);
}
```
