## 接口描述

`GET /future/trade/v1/entrust/profit-detail`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称     | 类型    | 必填 | 默认值 | 描述          |
| -------- | ------- | ---- | ------ | ------------- |
| profitId | integer | true | N/A    | Stop limit ID |
