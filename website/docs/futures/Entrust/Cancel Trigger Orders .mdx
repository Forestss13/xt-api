## 接口描述

`POST /future/trade/v1/entrust/cancel`

#### **Limit Flow Rules**

200/s/apikey

## 参数

| 参数名    | 类型   | 是否必填 | 默认值 | 描述             |
| --------- | ------ | -------- | ------ | ---------------- |
| entrustId | string | true     | N/A    | Trigger Order ID |

## 响应示例

```json
{
  "error": {"code": "", "msg": ""},
  "msgInfo": "",
  "result": true,
  "returnCode": 0
}
```
