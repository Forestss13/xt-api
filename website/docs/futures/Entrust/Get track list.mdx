### Description

**GET** `/future/trade/v1/entrust/track-list`

### Limit Flow Rules

200/s/apikey

### Parameters

| Name      | Type    | Mandatory | Default | Description               |
| --------- | ------- | --------- | ------- | ------------------------- |
| page      | integer | false     | 1       | Page                      |
| size      | integer | false     | 10      | Quantity of a single page |
| endTime   | integer | false     | N/A     | End time                  |
| startTime | integer | false     | N/A     | Start time                |
| symbol    | string  | false     | N/A     | Symbol                    |

### Example Code

```java
public void getTrackDetail() {
    String text = HttpUtil.get(URL + "/data/api/future/trade/v1/entrust/track-list");
    System.out.println(text);
}
```
