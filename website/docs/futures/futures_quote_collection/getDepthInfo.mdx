## 接口

**GET** `/future/market/v1/public/cg/orderbook`

Note：This method does not require a signature.

## 请求参数

| 名称     | 类型   | 必填 | 默认值 | 说明                      | 范围  |
| -------- | ------ | ---- | ------ | ------------------------- | ----- |
| `symbol` | string | 是   | N/A    | Trading pair, eg.BTC-USDT |       |
| `level`  | int    | 否   | 50     |                           | 1-200 |

## 示例请求

### Java

```java
public void getMarketConfig() {
	String text = HttpUtil.get(URL + "/data/api/future/market/v1/public/cg/orderbook");
	System.out.println(text);
}
```
