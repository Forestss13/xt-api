### Response Message Format

This section outlines the response message format when interacting with WebSocket APIs.

### Response Format

```javascript
{
    "id": "{id}",   // Call back ID
    "code": 1,      // Result 0=success; 1=fail; 2=listenKey invalid
    "msg": ""
}
```
