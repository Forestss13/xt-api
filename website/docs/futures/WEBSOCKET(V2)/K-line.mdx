### K-line

**Request**

Format: `kline@{symbol},{interval}`

Available intervals: `1m`, `3m`, `5m`, `15m`, `30m`, `1h`, `2h`, `4h`, `6h`, `8h`, `12h`, `1d`, `3d`, `1w`, `1M`

Example: `kline@btc_usdt,5m`

Rate: 1000ms

---

### Python Example

```python
import websocket

# Open a WebSocket connection
ws = websocket.create_connection("wss://fstream.xt.com/ws/market")
ws.send('{"method":"SUBSCRIBE","params":["kline@btc_usdt,5m"],"id":1}')


{
  "topic": "kline",
  "event": "kline@btc_usdt,5m",
  "data": {
    "s": "btc_index",  // Trading pair
    "o": "49000",      // Opening price
    "c": "50000",      // Closing price
    "h": "0.1",        // Highest price
    "l": "0.1",        // Lowest price
    "a": "0.1",        // Volume
    "v": "0.1",        // Turnover
    "ch": "0.21",      // Quote change
    "t": 123124124     // Timestamp
  }
}
```
