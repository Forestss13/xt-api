subscribe:

```js
  {
     "method": "SUBSCRIBE/UNSUBSCRIBE",
     "params": [
         "{order}@{listenKey}",
         "{trade}@{listenKey}",
         "{balance}@{listenKey}",
         "{position}@{listenKey}",
         "{notify}@{listenKey}"
      ],
     "id": "{id}"    //user defined
  }
```

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |
