subscribe:

```js
  {
     "method": "SUBSCRIBE/UNSUBSCRIBE",
     "params": [
         "{position}@{listenKey}",
      ],
     "id": "{id}"
  }
```

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

           "positionSide": "LONG",
           "positionSize":"123",  // Position quantity

           "availableCloseSize": "23",  //  Available quantity (Cont)

           "entryPrice":"213",   //  Open position average price

           "underlyingType": ""// COIN_BASED, U_BASED

       }
