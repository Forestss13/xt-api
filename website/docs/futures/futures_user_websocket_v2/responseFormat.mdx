## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

## 示例请求

### format

```javascript
{
    "id": "{id}",   //call back ID
    "code": 1,      //result 0=success;1=fail;2=listenKey invalid
    "msg": ""
}
```

## 更多示例

### Response-success

```json
{"id": "123", "code": 0, "msg": "success"}
```

### response-token invalid

```json
{"id": "123", "code": 401, "msg": "token expire"}
```
