```js
  {
     "method": "SUBSCRIBE/UNSUBSCRIBE",
     "params": [
         "{order}@{listenKey}",

     "id": "{id}"
  }
```

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |
