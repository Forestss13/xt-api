subscribe:

```js
  {
     "method": "SUBSCRIBE/UNSUBSCRIBE",
     "params": [
         "{balance}@{listenKey}",
      ],
     "id": "{id}"
  }
```

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |
