## 接口

**POST** `/future/trade/v1/order/cancel`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称      | 类型    | 必填 | 默认值 | 说明     | 范围 |
| --------- | ------- | ---- | ------ | -------- | ---- |
| `orderId` | Integer | 是   | N/A    | Order ID |      |

## 响应示例

### Response

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": "", //Order ID
  "returnCode": 0
}
```
