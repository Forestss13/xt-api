## 接口

**POST** `/future/trade/v1/order/cancel-all`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | String | 是 | N/A | Trading pair (cancel all trading pair orders if pass "") |  |

{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": true,
  "returnCode": 0
}
