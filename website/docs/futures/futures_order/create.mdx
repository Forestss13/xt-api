## 接口

**POST** `/future/trade/v1/order/create`

#### **OrigQty Calculation Formula**

###### **Formula**

origQty = Truncate ((Balance _ Percent _ Leverage ) / (Mark_price \* Contract_size))

###### **Explain**

    Truncate : take the integer part

    Balance : (walletBalance - openOrderMarginFrozen) , api: /future/user/v1/compat/balance/list

    Percent : user input , exp: 0.2

    Leverage : leverage your want , exp: 20

    Mark_price : current symobl mark price , exp: 88888 (btc_usdt)

    Contract_size : contractSize , api: /future/market/v1/public/symbol/detail , Contract multiplier(face value)

###### **Example**

truncate(10000 _ 0.2 _ 20 / 88888 / 0.0001) = 4500

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `clientOrderId` | string | 否 | N/A | Client order ID |  |
| `symbol` | string | 是 |  | Trading pair |  |
| `orderSide` | string | 是 | N/A | Order side:BUY;SELL | BUY;SELL |
| `orderType` | string | 是 | N/A | Order type:LIMIT；MARKET | LIMIT；MARKET |
| `origQty` | number | 是 | N/A | Quantity (Cont) |  |
| `price` | number | 否 | N/A | Price |  |
| `timeInForce` | string | 否 | GTC | Valid way:GTC;IOC;FOK;GTX | GTC;IOC;FOK;GTX |
| `triggerProfitPrice` | number | 否 | N/A | Stop profit price |  |
| `triggerStopPrice` | number | 否 | N/A | Stop loss price |  |
| `positionSide` | string | 是 | N/A | Position side:LONG;SHORT | LONG;SHORT |
