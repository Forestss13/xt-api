## 接口

**POST** `/future/trade/v1/entrust/cancel-plan`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称        | 类型    | 必填 | 默认值 | 说明             | 范围 |
| ----------- | ------- | ---- | ------ | ---------------- | ---- |
| `entrustId` | integer | 是   | N/A    | Trigger order ID |      |
