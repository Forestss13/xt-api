## 接口

**GET** `/future/trade/v1/entrust/plan-detail`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称        | 类型    | 必填 | 默认值 | 说明     | 范围 |
| ----------- | ------- | ---- | ------ | -------- | ---- |
| `entrustId` | integer | 是   | N/A    | Order ID |      |

## 示例请求

### Java

```java
public void getMarketConfig() {
	String text = HttpUtil.get(URL + "/data/api/future/trade/v1/getMarketConfig");
	System.out.println(text);
}
```
