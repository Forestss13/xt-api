## 接口

**GET** `/future/trade/v1/entrust/track-list`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称        | 类型    | 必填 | 默认值 | 说明                      | 范围 |
| ----------- | ------- | ---- | ------ | ------------------------- | ---- |
| `page`      | integer | 否   | 1      | page                      |      |
| `size`      | integer | 否   | 10     | Quantity of a single page |      |
| `endTime`   | integer | 否   | N/A    |                           |      |
| `startTime` | integer | 否   | N/A    | start time                |      |
| `symbol`    | string  | 否   | N/A    | symbol                    |      |

## 示例请求

### Java

```java
public void getTrackDetail() {
	String text = HttpUtil.get(URL + "/data/api/future/trade/v1/entrust/track-list");
	System.out.println(text);
}
```
