## 接口

**GET** `/future/trade/v1/entrust/profit-detail`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称       | 类型    | 必填 | 默认值 | 说明          | 范围 |
| ---------- | ------- | ---- | ------ | ------------- | ---- |
| `profitId` | integer | 是   | N/A    | Stop limit ID |      |

## 示例请求

### Java

```java
public void getMarketConfig() {
	String text = HttpUtil.get(URL + "/data/api/future/trade/v1/getMarketConfig");
	System.out.println(text);
}
```
