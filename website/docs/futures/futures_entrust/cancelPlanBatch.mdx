## 接口

**POST** `/future/trade/v1/entrust/cancel-all-plan`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称     | 类型   | 必填 | 默认值 | 说明                        | 范围 |
| -------- | ------ | ---- | ------ | --------------------------- | ---- |
| `symbol` | string | 是   |        | Trading pair. e.g. btc_usdt |      |

## 示例请求

### Java

```java
public void getKLine() {
	String text = HttpUtil.get(URL + "/data/api/future/trade/v1/getKLine?market=btc_usdt&type=1min&since=0");
	System.out.println(text);
}
```
