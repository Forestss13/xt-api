## 接口

**GET** `/future/trade/v1/entrust/plan-list-history`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 是 | N/A | Trading pairs (queries all trading pairs if not passed) |  |
| `direction` | string | 否 | NEXT | Direction（PREV:Previous page；NEXT:Next page） | PREV;NEXT |
| `id` | integer | 否 | N/A | id |  |
| `limit` | integer | 否 | 10 | Limit |  |
| `startTime` | integer | 否 | N/A | Start time |  |
| `endTime` | integer | 否 | N/A | End time |  |

## 示例请求

### Java

```java
public void getMarketConfig() {
```
