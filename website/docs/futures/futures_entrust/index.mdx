# 期货委托接口

期货交易委托相关接口提供了完整的期货交易委托管理功能，包括创建、取消、查询等操作。

## 主要功能

### 📋 委托管理

- 创建追踪委托
- 取消单个追踪委托
- 取消所有追踪委托
- 查询委托详情

### 📊 计划委托

- 创建计划委托
- 取消计划委托
- 批量取消计划委托
- 查询计划委托

### 💰 止盈止损

- 创建止盈止损
- 取消止盈止损
- 批量取消止盈止损
- 修改止盈止损

## 接口列表

### 追踪委托相关

- [创建追踪委托](./createTrack) - 创建新的追踪委托订单
- [取消单个追踪委托](./cancelTrack) - 取消指定的追踪委托
- [取消所有追踪委托](./cancelAllTrack) - 取消所有活跃的追踪委托
- [查询追踪委托详情](./getTrackDetail) - 获取追踪委托的详细信息
- [查询追踪委托列表](./getTrackList) - 获取所有追踪委托列表
- [查询追踪委托历史](./getTrackHistoryList) - 获取历史追踪委托记录

### 计划委托相关

- [创建计划委托](./createPlan) - 创建新的计划委托订单
- [取消计划委托](./cancelPlan) - 取消指定的计划委托
- [批量取消计划委托](./cancelPlanBatch) - 批量取消计划委托
- [查询计划委托](./getPlan) - 获取计划委托列表
- [根据 ID 查询计划委托](./getPlanById) - 根据 ID 获取计划委托详情
- [查询计划委托历史](./getPlanHistory) - 获取历史计划委托记录

### 止盈止损相关

- [创建止盈止损](./createProfit) - 创建新的止盈止损订单
- [取消止盈止损](./cancelProfit) - 取消指定的止盈止损
- [批量取消止盈止损](./cancelProfitBatch) - 批量取消止盈止损
- [修改止盈止损](./updateProfit) - 修改止盈止损参数
- [查询止盈止损](./getProfit) - 获取止盈止损列表
- [根据 ID 查询止盈止损](./getProfitById) - 根据 ID 获取止盈止损详情

## 注意事项

1. **权限要求**：所有委托接口都需要有效的 API 密钥和签名
2. **频率限制**：大部分接口限制为 200 次/秒/API 密钥
3. **参数验证**：请确保传入的参数符合接口要求
4. **错误处理**：请根据返回的错误码进行相应的错误处理

## 快速开始

1. 首先确保您已经申请了 API 密钥
2. 了解[基础信息](../futures_documentation/apiBasicInfo)和[签名算法](../futures_documentation/signStatement)
3. 查看具体接口的请求参数和响应格式
4. 在测试环境中验证接口调用

更多详细信息请参考各个具体接口的文档。
