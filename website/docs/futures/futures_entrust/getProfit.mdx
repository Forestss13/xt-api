## 接口

**GET** `/future/trade/v1/entrust/profit-list`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 是 | N/A | Trading pairs (queries all trading pairs if not passed) |  |
| `page` | integer | 否 | 1 | Page |  |
| `size` | integer | 否 | 10 | Quantity of a single page |  |
| `startTime` | integer | 否 | N/A | Start time |  |
| `endTime` | integer | 否 | N/A | End time |  |
| `state` | string | 是 | N/A | Order state:NOT_TRIGGERED：New order (not triggered);TRIGGERING:Triggering;TRIGGERED:Triggered;USER_REVOCATION:User revocation;PLATFORM_REVOCATION:Platform revocation (rejection);EXPIRED:expired;UNFINISHED:Unfinished;HISTORY:(History) | NOT_TRIGGERED;TRIGGERING;TRIGGERED;USER_REVOCATION;PLATFORM_REVOCATION;EXPIRED;UNFINISHED;HISTORY |
