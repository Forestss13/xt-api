## 接口

**POST** `/future/trade/v1/entrust/create-plan`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `clientOrderId` | string | 否 | N/A | Client order ID |  |
| `symbol` | string | 是 |  | Trading pair |  |
| `orderSide` | string | 是 | N/A | Order side:BUY;SELL | BUY;SELL |
| `entrustType` | string | 是 | N/A | Order type:TAKE_PROFIT(Take Profit Limit Order);STOP(Stop Limit Order);TAKE_PROFIT_MARKET(Take Profit Market Order);STOP_MARKET(Stop Loss Market Order) | TAKE_PROFIT;STOP;TAKE_PROFIT_MARKET;STOP_MARKET |
| `origQty` | number | 是 | N/A | Quantity (Cont) |  |
| `price` | number | 否 | N/A | Price |  |
| `stopPrice` | number | 是 | N/A | Trigger price |  |
| `timeInForce` | string | 是 | N/A | Valid way:GTC;IOC;FOK;GTX, Market orders only support IOC | GTC;IOC;FOK;GTX |
| `triggerPriceType` | string | 是 | N/A | Trigger price type:INDEX_PRICE(Index price)；MARK_PRICE(Mark price)；LATEST_PRICE(latest price) | INDEX_PRICE;MARK_PRICE;LATEST_PRICE |
| `positionSide` | string | 是 | N/A | Position side:LONG;SHORT | LONG;SHORT |
