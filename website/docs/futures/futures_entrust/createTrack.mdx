## 接口

**POST** `/future/trade/v1/entrust/create-track`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `callback` | string | 是 | N/A | Callback range configuration | FIXED;PROPORTION |
| `callbackVal` | number | 是 | N/A | Callback value | Greater than 0 |
| `orderSide` | string | 是 | N/A | Order side | BUY;SELL |
| `origQty` | number | 是 | N/A | Original quantity(count) |  |
| `positionSide` | string | 是 | N/A | Position side:LONG;SHORT | BOTH;LONG;SHORT |
| `positionType` | string | 是 | N/A | Position type | CROSSED;ISOLATED |
| `symbol` | string | 是 | N/A | Trading pair |  |
| `triggerPriceType` | string | 是 | N/A | Trigger price type:INDEX_PRICE(Index price)；MARK_PRICE(Mark price)；LATEST_PRICE(latest price) | INDEX_PRICE;MARK_PRICE;LATEST_PRICE |
| `activationPrice` | number | 否 | N/A | Activation price |  |
| `clientMedia` | string | 否 | N/A | Client media |  |
| `clientMediaChannel` | string | 否 | N/A | Client media channel |  |
| `clientOrderId` | string | 否 | N/A | client order id |  |
| `expireTime` | integer | 否 | N/A | expire time |  |
