## 接口

**POST** `/future/trade/v1/entrust/update-profit-stop`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称                 | 类型    | 必填 | 默认值 | 说明             | 范围 |
| -------------------- | ------- | ---- | ------ | ---------------- | ---- |
| `profitId`           | integer | 是   | N/A    | Stop limit ID    |      |
| `triggerProfitPrice` | number  | 否   | N/A    | TP trigger price |      |
| `triggerStopPrice`   | number  | 否   | N/A    | SL trigger price |      |

## 响应示例

{
  "error": {
    "code": "",

  },

  "returnCode": 0
}
