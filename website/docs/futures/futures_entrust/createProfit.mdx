## 接口

**POST** `/future/trade/v1/entrust/create-profit`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 否 | N/A | Trading pair |  |
| `origQty` | integer | 是 |  | Quantity (Cont) |  |
| `triggerProfitPrice` | integer | 是 |  | TP trigger price |  |
| `triggerStopPrice` | integer | 是 |  | SL trigger price |  |
| `expireTime` | integer | 是 |  | Expiration time |  |
| `positionSide` | string | 是 |  | Position side:LONG;SHORT | LONG;SHORT |
