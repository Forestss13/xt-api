## 接口

**GET** `/future/trade/v1/entrust/track-list-history`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `direction` | string | 否 | NEXT | Direction（PREV:Previous page；NEXT:Next page） | PREV;NEXT |
| `limit` | integer | 否 | 10 | Limit |  |
| `id` | integer | 否 | N/A |  |  |
| `endTime` | integer | 否 | N/A | End time |  |
| `startTime` | integer | 否 | N/A | start time |  |
| `symbol` | string | 否 | N/A | symbol |  |

## 示例请求

### Java

```java
public void getTrackDetail() {
	String text = HttpUtil.get(URL + "/data/api/future/trade/v1/entrust/track-list-history");
	System.out.println(text);
}
```
