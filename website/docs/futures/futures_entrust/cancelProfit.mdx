## 接口

**POST** `/future/trade/v1/entrust/cancel-profit-stop`

#### **Limit Flow Rules**

200/s/apikey

## 请求参数

| 名称       | 类型    | 必填 | 默认值 | 说明          | 范围 |
| ---------- | ------- | ---- | ------ | ------------- | ---- |
| `profitId` | integer | 是   | N/A    | Stop limit ID |      |

## 示例请求

```java
public void getKLine() {
```
