## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

## 示例请求

### format

```javascript
{
    "topic": "trade",
    "event": "trade@btc_usdt",    //title
    "data": { }
}
```

## 更多示例
