**request**

format: ticker@\{symbol\}

eg: ticker@btc_usdt

rate: 1000ms

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

## 响应示例

### Response

```json
{
  "topic": "ticker",
  "event": "ticker@btc_usdt",
  "data": {
    "s": "btc_index", //trading pair
    "o": "49000", // opening price
    "c": "50000", //Closing price
    "h": "0.1", //highest price
    "l": "0.1", //lowest price
    "a": "0.1", //volume
    "v": "0.1", //Turnover
    "ch": "0.21", //Quote change
    "t": 123124124 //timestamp
  }
}
```
