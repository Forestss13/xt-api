**request**

format: depth_update@\{symbol\},\{interval\}

interval: 100/250/500/1000 &nbsp; default rate 100ms

eg: depth_update@btc_usdt,100ms

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

## 响应示例

### Response

```json
{
  "topic": "depth_update",
  "event": "depth_update@btc_usdt",
  "data": {
    "s": "btc_usdt", // symbol
    "pu": 120, // previousUpdateId = previous lastUpdateId
    "fu": 121, // firstUpdateId = previous lastUpdateId + 1
    "u": 123, // lastUpdateId
    "a": [
      // asks  sell order
      [
        // [0]price, [1]quantity
        "34000", //price
        "1.2" //quantity
      ],
      ["34001", "2.3"]
    ],
    "b": [
      // bids buy order
      ["32000", "0.2"],
      ["31000", "0.5"]
    ],
    "t": 123456789 // time stamp
  }
}
```
