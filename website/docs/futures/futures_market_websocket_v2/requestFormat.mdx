## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

## 示例请求

### subscribe

```javascript
{
    "method": "subscribe",
    "params": [
        "{topic}@{arg},{arg}",
        "{topic}@{arg}"
    ],
    "id": "{id}"    //call back ID
}
```

### unsubscribe

```javascript
{
    "method": "unsubscribe",
    "params": [
        "{topic}@{arg},{arg}"
    ],
    "id": "{id}"   //call back ID
}
```
