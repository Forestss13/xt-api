**request**

&nbsp;

format: kline@\{symbol\},\{interval\}

interval: 1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 8h, 12h, 1d, 3d, 1w, 1M

eg: kline@btc_usdt,5m

rate: 1000ms

&nbsp;

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

{

    "o": "49000", // opening price
    "c": "50000", //Closing price

    "l": "0.1", //lowest price
    "a": "0.1", //volume
    "v": "0.1", //Turnover
    "ch": "0.21", //Quote change
    "t": 123124124 //timestamp
  }
}
