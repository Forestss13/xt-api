**request**

format: fund_rate@\{symbol\}

eg: fund_rate@btc_usdt

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |
