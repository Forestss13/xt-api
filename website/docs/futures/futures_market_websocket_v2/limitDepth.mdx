**request**

&nbsp;

format: depth@\{symbol\},\{levels\},\{interval\}

levels: 5/10/20/50

interval: 100/250/500/1000 &nbsp; default rate 1000ms

eg: depth@btc_usdt,50,1000ms

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |
