**request**

format: agg_ticker@\{symbol\}

eg: agg_ticker@btc_usdt

rate: 1000ms

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

### Response

```json
{
  "topic": "agg_ticker",
  "event": "agg_ticker@btc_usdt",
  "data": {
    "s": "btc_index", //trading pair
    "o": "49000", // opening price
    "c": "50000", //Closing price
    "h": "0.1", //highest price
    "l": "0.1", //lowest price
    "a": "0.1", //volume
    "v": "0.1", //Turnover
    "ch": "0.21", //Quote change

    "i": "0.21", //index price

    "m": "0.21", //mark price

    "bp": "0.21", //bid price

    "ap": "0.21", //ask price
    "t": 123124124 //timestamp
  }
}
```
