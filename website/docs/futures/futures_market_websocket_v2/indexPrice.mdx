**request**

format: index_price@\{symbol\}

eg: index_price@btc_usdt

rate: 1000ms

## 请求参数

| 名称   | 类型   | 必填 | 默认值 | 说明 | 范围 |
| ------ | ------ | ---- | ------ | ---- | ---- |
| `None` | string | 否   |        |      |      |

## 响应示例

### Response

```json
{
  "topic": "index_price",
  "event": "index_price@btc_usdt",
  "data": {
    "s": "btc_usdt", //trading pair
    "p": "50000", //price
    "t": 123124124 //timestamp
  }
}
```
