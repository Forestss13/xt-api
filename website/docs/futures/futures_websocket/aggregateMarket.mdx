Subscribe：

\{"channel":"ex_single_market","market":\{Market\},"event":"addChannel"\}

Unsubscribe：

\{"channel":"ex_single_market","market":\{Market\},"event":"removeChannel"\}

## 请求参数

| 名称     | 类型   | 必填 | 默认值 | 说明            | 范围                  |
| -------- | ------ | ---- | ------ | --------------- | --------------------- |
| `market` | string | 是   | N/A    | Trading markets | btc_usdt, eth_usdt... |
