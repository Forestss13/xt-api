## 接口

**GET** `/future/market/v1/public/q/ticker/book`

Note：This method does not require a signature.

## 请求参数

| 名称     | 类型   | 必填 | 默认值 | 说明         | 范围 |
| -------- | ------ | ---- | ------ | ------------ | ---- |
| `symbol` | string | 是   | N/A    | Trading pair |      |

## 示例请求

### Java

```java
public void getTickerBokk() {
	String text = HttpUtil.get(URL + "/data/api//future/market/v1/public/q/ticker/book?symbol=btc_usdt");
	System.out.println(text);
}
```

## 响应示例

### Response

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "ap": "", //ask price
    "aq": "", //ask amount
    "bp": "", //bid price
    "bq": "", //bid amount
    "s": "", //Trading pair
    "t": 0 //Time
  },
  "returnCode": 0
}
```
