## 接口

**GET** `/future/market/v1/public/contract/risk-balance`

Note：This method does not require a signature.

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 是 |  | Trading pair |  |
| `direction` | string | 否 | NEXT | Direction（PREV:Previous page；NEXT:Next page） | PREV;NEXT |
| `id` | integer | 否 | N/A | id |  |
| `limit` | integer | 否 | 10 | Limit |  |

## 示例请求

### Java

```java
public void getKLine() {
	String text = HttpUtil.get(URL + "/data/api/future/market/v1/getKLine?market=btc_usdt&type=1min&since=0");
	System.out.println(text);
}
```
