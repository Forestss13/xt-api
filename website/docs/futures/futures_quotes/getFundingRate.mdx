## 接口

**GET** `/future/market/v1/public/q/funding-rate`

#### **Limit Flow Rules**

`1/s/ip <br> Note：This method does not require a signature`.

## 请求参数

| 名称     | 类型   | 必填 | 默认值 | 说明         | 范围 |
| -------- | ------ | ---- | ------ | ------------ | ---- |
| `symbol` | string | 是   | N/A    | Trading pair |      |

## 示例请求

### Java

```java
public void getKLine() {
	String text = HttpUtil.get(URL + "/data/api/future/market/v1/getKLine?market=btc_usdt&type=1min&since=0");
	System.out.println(text);
}
```
