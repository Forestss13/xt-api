## 接口

**GET** `/future/market/v1/public/q/kline`

Note：This method does not require a signature.

## 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `symbol` | string | 是 | N/A | Trading pair |  |
| `interval` | string | 是 | N/A | Time-interval | 1m;5m;15m;30m;1h;4h;1d;1w |
| `startTime` | integer | 否 | N/A | Start time |  |
| `endTime` | integer | 否 | N/A | End time |  |
| `limit` | integer | 否 | N/A | Limit |  |

## 示例请求

### Java

```java
public void getKLine() {
	String text = HttpUtil.get(URL + "/data/api/v1/getKLine?market=btc_usdt&type=1min&since=0");
	System.out.println(text);
}
```
