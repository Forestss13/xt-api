## 接口

**GET** `/future/market/v1/public/q/ticker/books`

Note：This method does not require a signature.

## 示例请求

### Java

```java
public void getTickerBokk() {
	String text = HttpUtil.get(URL + "/data/api//future/market/v1/public/q/ticker/books?symbol=btc_usdt");
	System.out.println(text);
}
```

## 响应示例

### Response

```json
{
  "error": {
    "code": "",
    "msg": ""

  "msgInfo": "",

      "bp": "", //bid price

    }
  ],
  "returnCode": 0
}
```
