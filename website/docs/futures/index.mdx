# 期货交易 API

欢迎使用 XT 期货交易 API！本 API 提供了完整的期货交易功能，包括合约信息查询、下单交易、持仓管理等。

## 主要功能

- **合约管理**: 获取期货合约信息、价格、深度等
- **交易操作**: 开仓、平仓、调整杠杆等
- **风险控制**: 设置止损止盈、风险预警等
- **数据查询**: 历史订单、成交记录、资金流水等
