# API 基础信息

## 概述

XT API 现货交易接口提供完整的现货交易功能，包括市场数据、订单管理、账户信息等。

## 基础信息

- **基础 URL**: `https://api.xt.com`
- **协议**: HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8

## 认证方式

所有私有接口都需要进行签名认证：

1. **API Key**: 您的 API 密钥
2. **Secret Key**: 您的密钥
3. **签名算法**: HMAC SHA256

## 请求限制

- **频率限制**: 每秒最多 10 次请求
- **并发限制**: 最多 5 个并发连接
- **日限制**: 每日最多 100,000 次请求

## 响应格式

所有 API 响应都遵循统一的格式：

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    // 具体数据
  }
}
```

## 错误码

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 签名错误 |
| 1003 | 权限不足 |
| 1004 | 频率限制 |

## 开始使用

1. 申请 API 密钥
2. 查看接口文档
3. 实现签名算法
4. 开始调用接口
